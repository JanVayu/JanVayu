# नागरिक कृती साधने

JanVayu नागरिक, पत्रकार, संशोधक आणि वकिलांना हवा गुणवत्तेवर कृती करण्यासाठी व्यावहारिक साधने प्रदान करतो.

---

## RTI टेम्पलेट्स

यासाठी पूर्व-तयार माहिती अधिकार कायदा (2005) टेम्पलेट्स:

| टेम्पलेट | लक्ष्य प्राधिकरण |
|----------|-----------------|
| NCAP निधी वापर | राज्य प्रदूषण नियंत्रण मंडळ |
| GRAP अनुपालन अहवाल | Commission for Air Quality Management (CAQM) |
| स्रोत विभाजन अभ्यास | Central Pollution Control Board (CPCB) |
| औद्योगिक उत्सर्जन डेटा | राज्य PCB / MoEFCC |
| शहर कृती योजना स्थिती | महानगरपालिका |

टेम्पलेट्स **इंग्रजी आणि हिंदी** मध्ये उपलब्ध आहेत, [rtionline.gov.in](https://rtionline.gov.in) वरील RTI ऑनलाइन पोर्टलसाठी स्वरूपित.

---

## वकिली मार्गदर्शक

टप्प्याटप्प्याने मार्गदर्शक:

- **RTI कसा दाखल करायचा** — सादर करण्यापासून पहिल्या अपीलपर्यंत
- **तुमच्या वॉर्ड नगरसेवकाशी कसे संपर्क करायचा** — वॉर्ड-स्तरीय उत्तरदायित्व ब्रीफ जनरेटर वापरून
- **मीडियाशी कसे संवाद साधायचा** — डेटासह हवा गुणवत्तेची कथा तयार करणे
- **सार्वजनिक सुनावणीत कसे सहभागी व्हायचे** — पर्यावरणीय प्रभाव मूल्यांकनासाठी

---

## वॉर्ड-स्तरीय उत्तरदायित्व ब्रीफ

व्यासपीठाचा AI-सहाय्यित ब्रीफ जनरेटर (Google Gemini द्वारे चालणारा) कोणत्याही शहरासाठी सानुकूलित उत्तरदायित्व ब्रीफ तयार करतो, वॉर्ड नगरसेवक, रहिवासी कल्याण संघटना किंवा पत्रकारांसाठी बनवलेला. यात समाविष्ट आहे:

- शहरासाठी सध्याचे AQI रीडिंग्स
- NCAP लक्ष्यांशी तुलना
- प्रमुख स्थानिक प्रदूषण स्रोत
- निवडून आलेल्या अधिकाऱ्यांना विचारण्यासाठी सुचवलेले प्रश्न

ब्रीफ तयार करण्यासाठी, API endpoint वापरा: `GET /.netlify/functions/accountability-brief?city={cityKey}`

---

## मास्क निवड मार्गदर्शक

वेगवेगळ्या AQI पातळींसाठी योग्य मास्क निवडण्याचे व्यावहारिक मार्गदर्शक:

| AQI पातळी | शिफारस केलेले संरक्षण |
|-----------|----------------------|
| चांगला–मध्यम | मास्कची गरज नाही |
| संवेदनशील गटांसाठी अस्वास्थ्यकर | संवेदनशील व्यक्तींसाठी N95/KN95 |
| अस्वास्थ्यकर | सर्वांसाठी N95/KN95 शिफारसीय |
| अत्यंत अस्वास्थ्यकर | N95/KN95, बाहेरचा वेळ मर्यादित करा |
| धोकादायक | घरात राहा; बाहेर जाताना N95 |

## घरातील हवा गुणवत्ता

घरातील हवा गुणवत्ता विभागात हे समाविष्ट आहे:

- विविध इमारत प्रकारांसाठी वायुवीजन धोरणे
- एअर प्युरिफायर निवड (HEPA फिल्टर मानके, खोलीच्या आकारासाठी CADR रेटिंग)
- स्वयंपाकाचे इंधन आणि आरोग्य — LPG विरुद्ध घन इंधन, उज्ज्वला योजनेची पोहोच
- कमी किमतीच्या सेन्सर्ससह घरातील हवा गुणवत्ता मॉनिटरिंग
