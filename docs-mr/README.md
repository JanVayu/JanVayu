# JanVayu दस्तऐवज

**JanVayu** (जनवायु — "लोकांची हवा") हे भारताचे स्वतंत्र, नागरिकांनी चालवलेले हवा गुणवत्ता उत्तरदायित्व व्यासपीठ आहे. हे 30+ शहरांचा प्रदूषण डेटा रिअल-टाइम ट्रॅक करते, आरोग्यावरील परिणाम मोजते, NCAP बजेट खर्चावर लक्ष ठेवते आणि सरकारला जबाबदार धरते.

> **लाइव्ह व्यासपीठ:** [www.janvayu.in](https://www.janvayu.in)
> **GitHub:** [github.com/JanVayu/JanVayu](https://github.com/JanVayu/JanVayu)

---

## JanVayu म्हणजे काय?

भारतात वायुप्रदूषणामुळे दरवर्षी 1.7 ते 2 दशलक्ष लोकांचा मृत्यू होतो. PM2.5 प्रदर्शनामुळे भारतीयांचे सरासरी आयुर्मान 3.5 वर्षांनी कमी होते. दिल्लीचे वार्षिक PM2.5 ~100 µg/m³ आहे, जे WHO मार्गदर्शक तत्त्वापेक्षा 20 पट जास्त आहे. आर्थिक नुकसान वार्षिक $150 अब्जांपेक्षा जास्त आहे.

JanVayu ही कोणती मोहीम नाही. हा एक रेकॉर्ड आहे — हवा गुणवत्ता संकट, त्याचा डेटा, त्याचे बळी, त्याच्या धोरणे आणि त्याची सार्वजनिक स्मृती दस्तऐवजीकरण करणारे कायमस्वरूपी सार्वजनिक संग्रहालय.
