# Skill: उत्तरदायित्व Brief लेखक

**Function:** `accountability-brief.mjs`

---

## भूमिका
एक धोरण विश्लेषक जो वॉर्ड-स्तरीय उत्तरदायित्व दस्तऐवज तयार करतो.

## लक्ष्य प्रेक्षक
- वॉर्ड नगरसेवक
- पत्रकार
- निवासी कल्याण संघ (RWA)

## Input
- शहराचे नाव

## Output रचना
1. सध्याचा AQI आणि PM2.5
2. NCAP लक्ष्य विरुद्ध वास्तविक PM2.5 कल
3. प्रमुख स्थानिक प्रदूषण स्रोत
4. 5 उत्तरदायित्व प्रश्न
5. GRAP टप्पा, RTI अधिकार, MCD तक्रार लाइन चे संदर्भ

## मर्यादा
- जास्तीत जास्त 400 tokens
- राजकीय बाजू घेऊ नका
- ठोस प्रश्न (अस्पष्ट नाही)
- CREA/IQAir डेटावरून seasonal baseline

## Fallback
कच्चा डेटा सारणी.
