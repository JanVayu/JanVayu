# Skill: Visual Design

JanVayu चे UI/UX निर्णय आणि design तत्त्वे.

---

## Design तत्त्वे

1. **कार्यक्षमता आधी** — सौंदर्यापेक्षा उपयुक्तता
2. **Mobile-first** — बहुतांश भारतीय वापरकर्ते mobile वर
3. **कमी bandwidth** — किमान चित्रे, कोणताही video autoplay नाही
4. **सुलभ** — WCAG AA अनुपालन

## रंग पॅलेट

### Light Mode
- प्राथमिक: `#2563eb`
- पार्श्वभूमी: `#ffffff`
- Text: `#1e293b`

### Dark Mode
- पार्श्वभूमी: `#0f172a`
- Text: `#e2e8f0`

### AQI रंग
- चांगला: `#22c55e` (हिरवा)
- समाधानकारक: `#eab308` (पिवळा)
- मध्यम: `#f97316` (नारंगी)
- खराब: `#ef4444` (लाल)
- गंभीर: `#7c3aed` (जांभळा)
