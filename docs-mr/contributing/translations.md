## अनुवाद कसा करायचा

### Platform (index.html)
भाषा strings `index.html` मध्ये JS object म्हणून आहेत. नवीन भाषा जोडण्यासाठी:
1. विद्यमान भाषा object कॉपी करा
2. सर्व strings चे अनुवाद करा
3. भाषा toggle मध्ये पर्याय जोडा

### दस्तऐवजीकरण (GitBook)
हे दस्तऐवजीकरण `docs-mr/` directory मध्ये मराठी अनुवादासह आहे. इतर भाषांसाठी:
- `docs-hi/` — हिंदी
- `docs-ta/` — तमिळ
- `docs-bn/` — बंगाली

## अनुवाद मार्गदर्शक तत्त्वे

1. तांत्रिक शब्द इंग्रजीत ठेवा (AQI, PM2.5, API, इ.)
2. सोपी, बोलीभाषेचा वापर करा
3. कठीण संस्कृत शब्द टाळा
4. संक्षिप्त राहा — लांब वाक्ये बनवू नका
