# Claude Code — JanVayu कैसे बनाया गया

JanVayu का विकास **Claude Code** — Anthropic के आधिकारिक CLI एजेंट — की सहायता से हुआ।

---

## Claude Code क्या है?

Claude Code एक टर्मिनल-आधारित AI कोडिंग एजेंट है। आप इसे अपनी प्रोजेक्ट डायरेक्टरी में चलाते हैं और यह:
- फ़ाइलें पढ़ और लिख सकता है
- शेल कमांड चला सकता है
- कोडबेस खोज सकता है
- कमिट और पुल रिक्वेस्ट बना सकता है
- जटिल मल्टी-फ़ाइल रिफैक्टर संभाल सकता है

## मुख्य शक्तियाँ

1. **पूर्ण कोडबेस संदर्भ** — Claude आपका कोड पढ़ता है लिखने से पहले
2. **बाधा-प्रथम** — "कोई फ्रेमवर्क नहीं" जैसी बाधाएँ समझता है
3. **बहु-फ़ाइल समन्वय** — एक प्रॉम्प्ट से Functions, HTML, CSS, JS सब बदलता है
4. **Git-नेटिव** — ब्रांच, कमिट, PR एक ही टूल में
