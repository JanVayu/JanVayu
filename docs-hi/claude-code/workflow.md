# Claude Code डेवलपमेंट वर्कफ़्लो

JanVayu के निर्माण में उपयोग किया गया दैनिक वर्कफ़्लो।

---

## सुविधा विकास

### चरण 1: सुविधा परिभाषित करें
```
> एक विसंगति पहचान बैनर जोड़ें जो दिल्ली, मुंबई, कोलकाता,
> चेन्नई और बेंगलुरु के लिए PM2.5 की मौसमी बेसलाइन से तुलना करे
```

### चरण 2: Claude कोडबेस पढ़ता है
- `index.html` — मौजूदा UI पैटर्न
- मौजूदा Netlify Functions — कोड शैली
- `package.json` — उपलब्ध डिपेंडेंसी

### चरण 3: Claude कार्यान्वयन करता है
- `netlify/functions/anomaly-check.mjs` बनाता है
- `index.html` में बैनर HTML/CSS/JS जोड़ता है

### चरण 4: समीक्षा और सुधार
```
> ब्रीफ़ बहुत लंबी है। Gemini आउटपुट 400 टोकन तक सीमित करें।
```

### चरण 5: कमिट और PR
Claude `git add`, `git commit`, `git push`, और `gh pr create` चलाता है।

## बग फ़िक्स वर्कफ़्लो

```
> anomaly-check फ़ंक्शन WAQI डाउन होने पर 500 लौटा रहा है।
> इसे कैश्ड डेटा के आधार पर 200 लौटाना चाहिए।
```

Claude न्यूनतम बदलाव करता है — पूरे फ़ंक्शन को दोबारा नहीं लिखता।
