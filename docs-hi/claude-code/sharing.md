# साझाकरण एवं प्रतिकृति

JanVayu का Claude Code वर्कफ़्लो कैसे साझा करें या अपने प्रोजेक्ट के लिए दोहराएँ।

---

## टीम के लिए

### नए योगदानकर्ता को शामिल करें
1. रिपो क्लोन करें
2. यह Docsify प्रलेखन पढ़ें
3. Claude Code इंस्टॉल करें: `npm install -g @anthropic-ai/claude-code`
4. सत्र शुरू करें: `cd JanVayu && claude`

---

## अपने प्रोजेक्ट के लिए

### चरण 1: बाधा-प्रथम दृष्टिकोण अपनाएँ
- परिभाषित करें कि प्रोजेक्ट क्या *नहीं* उपयोग करेगा
- `CLAUDE.md` फ़ाइल में लिखें

### चरण 2: स्किल फ़ाइलें बनाएँ
AI सुविधाओं के लिए संरचित सिस्टम प्रॉम्प्ट दस्तावेज़ करें।

### चरण 3: Git Hooks सेटअप करें
JanVayu के `.githooks/` कॉपी करें।

---

## JanVayu को दूसरे शहर के लिए फ़ोर्क करना

JanVayu MIT-लाइसेंस है:
1. GitHub पर फ़ोर्क करें
2. `index.html` में शहर सूची अपडेट करें
3. `anomaly-check.mjs` में मौसमी बेसलाइन बदलें
4. WAQI स्टेशन ID अपडेट करें
5. स्थानीय भाषाओं में अनुवाद करें
6. अपनी Netlify साइट सेटअप करें

Claude Code इन सभी चरणों में मदद कर सकता है।
