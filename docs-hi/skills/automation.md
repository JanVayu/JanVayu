# स्किल: स्वचालन

JanVayu के अनुसूचित कार्य और स्वचालित प्रक्रियाएँ।

---

## अनुसूचित कार्य

### फ़ीड रिफ्रेश (हर 4 घंटे)
- **फ़ंक्शन:** `scheduled-fetch.mjs`
- Reddit, Twitter/X, Instagram, समाचार फ़ेच
- Netlify Blobs में कैश
- विफलता पर पुराना कैश बनाए रखता है

### दैनिक डाइजेस्ट (8 AM IST)
- **फ़ंक्शन:** `daily-digest.mjs`
- सदस्यों के शहरों का AQI फ़ेच
- HTML ईमेल तैयार
- Resend API से भेजना

---

## स्वचालित जाँच

### विसंगति पहचान
- पेज लोड पर ट्रिगर
- 5 शहरों की मौसमी बेसलाइन से तुलना
- 10-मिनट कैश

### लिंक जाँच (CI)
- GitHub Actions (push/PR पर)
- Lychee लिंक चेकर
- सोशल मीडिया और localhost URLs को बाहर रखता है

### Dependabot
- मासिक npm अपडेट जाँच
- GitHub Actions अपडेट जाँच

## निगरानी

### फ़ीड स्वास्थ्य
- `feed-status.js` — प्रत्येक फ़ीड की ताज़गी जाँच

- सभी फ़ंक्शन `console.log` से त्रुटियाँ लॉग करते हैं
