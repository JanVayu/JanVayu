# स्किल: विज़ुअल डिज़ाइन

JanVayu के UI/UX निर्णय और डिज़ाइन सिद्धांत।

---

## डिज़ाइन सिद्धांत

1. **कार्यक्षमता पहले** — सुंदरता से पहले उपयोगिता
2. **मोबाइल-फर्स्ट** — अधिकांश भारतीय उपयोगकर्ता मोबाइल पर
3. **कम बैंडविड्थ** — न्यूनतम चित्र, कोई वीडियो ऑटोप्ले नहीं
4. **सुलभ** — WCAG AA अनुपालन
