# स्किल्स एवं AI प्रॉम्प्ट्स

यह अनुभाग AI स्किल फ़ाइलों — सिस्टम प्रॉम्प्ट, निर्देश पैटर्न, और प्रॉम्प्ट इंजीनियरिंग निर्णयों — का दस्तावेज़ीकरण करता है।

---

## स्किल फ़ाइल क्या है?

एक संरचित सिस्टम प्रॉम्प्ट जो AI मॉडल को बताती है कि एक विशिष्ट संदर्भ में कैसे व्यवहार करना है:
- मॉडल की **भूमिका और व्यक्तित्व**
- प्राप्त होने वाला **डेटा**
- **आउटपुट प्रारूप**
- **स्वर, लंबाई, और दावों पर बाधाएँ**
- बचने योग्य **विफलता मोड**

## सामान्य प्रॉम्प्टिंग सिद्धांत

1. **वास्तविक डेटा पर आधारित** — मॉडल कभी स्मृति से उत्तर नहीं देता
2. **सटीक आउटपुट प्रारूप** — विशेष रूप से जवाबदेही ब्रीफ़ के लिए
3. **लंबाई सीमित** — हर प्रॉम्प्ट में स्पष्ट शब्द/टोकन सीमा
4. **विफलता मोड नामित** — "सामान्य सलाह न दें"
5. **बहुभाषी** — "प्रश्न की भाषा में उत्तर दें"
6. **ग्रेसफुल फ़ॉलबैक** — हर फ़ंक्शन में non-AI विकल्प
