AI सहायता से JanVayu कोडबेस बनाने में उपयोग किए गए प्रॉम्प्ट पैटर्न।

---

## मूल सिद्धांत: बाधा-प्रथम प्रॉम्प्टिंग

> "Vanilla JavaScript only. No frameworks. No npm imports in the frontend. No build step."

बिना इस बाधा के, AI React components और Webpack configs देगा।

---

## प्रॉम्प्ट पैटर्न

### 1. "नागरिक डेटा" पैटर्न
```
एक [सुविधा] लिखें। डेटा [स्रोत] से आता है।
Plain HTML/CSS/JS में, कोई फ्रेमवर्क नहीं।
दर्शक 2G कनेक्शन पर हो सकते हैं।
लोड गति को प्राथमिकता दें।
```

### 2. "सर्वरलेस फ़ंक्शन" पैटर्न
```
Netlify Function (ES module, .mjs) लिखें:
- CORS प्रीफ़्लाइट हैंडल करें
- JSON लौटाएँ
- try/catch हर बाहरी कॉल पर
- सीक्रेट process.env से पढ़ें
- ग्रेसफुल फ़ॉलबैक शामिल करें
```

### 3. "डीबगिंग" पैटर्न
```
यह फ़ंक्शन [त्रुटि] लौटा रहा है।
फ़ंक्शन दोबारा न लिखें।
समस्या की विशिष्ट पंक्ति पहचानें।
न्यूनतम बदलाव प्रस्तावित करें।
```
