## प्रमुख डिज़ाइन निर्णय

### एकल HTML फ़ाइल
पूरा फ्रंट-एंड `index.html` में है — इनलाइन CSS और JavaScript, कोई बिल्ड स्टेप नहीं, कोई बंडलर नहीं, कोई फ्रेमवर्क नहीं। यह बुनियादी HTML/JS कौशल वाले योगदानकर्ताओं के लिए कोडबेस को सुलभ बनाता है।

### सर्वर-साइड प्रॉक्सीइंग
सोशल मीडिया और समाचार API को Netlify Functions के माध्यम से फ़ेच किया जाता है — CORS समस्याओं से बचने और API कुंजियों की सुरक्षा के लिए।

### Blob कैशिंग
`scheduled-fetch.mjs` हर 4 घंटे चलता है और सभी फ़ीड डेटा Netlify Blobs में लिखता है। ऑन-डिमांड फ़ंक्शन कैश से तुरंत सर्व करते हैं।

### क्लाइंट-साइड AQI
WAQI API सीधे ब्राउज़र से हर 10 मिनट में कॉल होता है। टोकन एक फ्री-टियर पब्लिक कुंजी है।

### कोई फ्रेमवर्क, कोई बिल्ड स्टेप नहीं
कोई `npm run build` नहीं, कोई Webpack नहीं, कोई React नहीं। डिप्लॉय आर्टिफैक्ट रिपॉजिटरी ही है।
