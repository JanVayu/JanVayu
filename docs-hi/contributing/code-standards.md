# कोड मानक

JanVayu कोडबेस के लिए कोडिंग मानक।

---

## सामान्य नियम

- **Vanilla JavaScript** — कोई फ्रेमवर्क नहीं
- **ES2020** — नए फ़ीचर नहीं (ब्राउज़र संगतता)
- **कोई बिल्ड स्टेप नहीं**
- **कोई npm डिपेंडेंसी क्लाइंट पर नहीं**

---

## HTML
- सिमेंटिक, सुलभ मार्कअप
- ARIA roles जहाँ आवश्यक
- सभी इमेज में alt text

## CSS
- CSS Custom Properties थीमिंग के लिए
- मोबाइल-फर्स्ट रिस्पॉन्सिव
- WCAG AA रंग कंट्रास्ट

## JavaScript
- Fetch API (axios नहीं)
- DOM API (jQuery नहीं)
- try/catch हर बाहरी कॉल पर
- console.log डीबगिंग के लिए (throw नहीं)

## Netlify Functions
- CORS प्रीफ़्लाइट हैंडल करें
- JSON लौटाएँ
- process.env से सीक्रेट पढ़ें
- ग्रेसफुल फ़ॉलबैक अनिवार्य
