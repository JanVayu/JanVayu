# कैसे योगदान दें

JanVayu एक ओपन सोर्स प्रोजेक्ट है। आपका योगदान स्वागत है!

---

## योगदान के तरीके

### 1. कोड योगदान
- बग फ़िक्स
- नई सुविधाएँ
- प्रदर्शन सुधार
- पहुँच (accessibility) सुधार

### 2. सामग्री योगदान
- डेटा स्रोत जोड़ना
- नीति अपडेट
- शोध संदर्भ

### 3. अनुवाद
- मौजूदा भाषाओं में सुधार
- नई भाषाएँ जोड़ना

### 4. बग रिपोर्ट
- GitHub Issues में रिपोर्ट करें
- Issue template का उपयोग करें

---

## प्रक्रिया

1. रिपो फ़ोर्क करें
2. फ़ीचर ब्रांच बनाएँ (`git checkout -b feature/my-feature`)
3. बदलाव करें
4. `netlify dev` से लोकली टेस्ट करें
5. कमिट करें (सम्मेलन का पालन करें)
6. पुल रिक्वेस्ट बनाएँ

---

## कमिट संदेश सम्मेलन

हर कमिट इन उपसर्गों से शुरू होना चाहिए:

| उपसर्ग | उपयोग |
|--------|-------|
| `Add` | नई सुविधा |
| `Fix` | बग फ़िक्स |
| `Update` | मौजूदा सुविधा में सुधार |
| `Translate` | अनुवाद |
| `Docs` | प्रलेखन |
| `Refactor` | कोड पुनर्गठन |

---

## लोकल सेटअप

```bash
git clone https://github.com/YOUR-USERNAME/JanVayu.git
cd JanVayu
npm install
netlify dev
```
