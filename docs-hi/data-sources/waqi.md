# रियल-टाइम AQI (WAQI)

WAQI (World Air Quality Index) API JanVayu का प्राथमिक रियल-टाइम डेटा स्रोत है।

---

## WAQI क्या है?

WAQI परियोजना दुनिया भर के 12,000+ मॉनिटरिंग स्टेशनों से रियल-टाइम वायु गुणवत्ता डेटा एकत्रित करती है।

---

## JanVayu में उपयोग

- **500+ भारतीय स्टेशन** कवर
- **हर 10 मिनट** में ऑटो-रिफ्रेश
- **क्लाइंट-साइड** कॉल (सीधे ब्राउज़र से)
- **फ्री-टियर** पब्लिक टोकन

## उपलब्ध डेटा
- AQI (Air Quality Index)
- PM2.5 (µg/m³)
- PM10, NO2, SO2, CO, O3
- तापमान, आर्द्रता, हवा की गति

## API एंडपॉइंट
- `api.waqi.info/feed/{city}/` — एकल शहर
- `api.waqi.info/map/bounds/` — भौगोलिक सीमा के भीतर स्टेशन
