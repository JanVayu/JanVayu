# फ्रंटएंड स्टैक

## HTML/CSS/JavaScript (Vanilla)

पूरा फ्रंटएंड एक `index.html` फ़ाइल है — ~11,300 पंक्तियाँ — इनलाइन `<style>` और `<script>` ब्लॉक के साथ।

---

## CSS आर्किटेक्चर

- **CSS Custom Properties** थीमिंग के लिए (लाइट/डार्क मोड)
- **कोई प्रीप्रोसेसर नहीं** (Sass, Less, PostCSS नहीं)
- **मोबाइल-फर्स्ट** रिस्पॉन्सिव डिज़ाइन
- **WCAG AA** रंग कंट्रास्ट अनुपालन

---

## Chart.js
- AQI ट्रेंड चार्ट
- स्वास्थ्य प्रभाव विज़ुअलाइज़ेशन
- मेट्रो बनाम क्षेत्रीय तुलना
- CDN से लोड (~60 KB gzipped)

## Leaflet.js + OpenStreetMap
- 40+ शहरों का इंटरैक्टिव मानचित्र
- रंग-कोडित AQI मार्कर
- निःशुल्क और ओपन सोर्स
- CDN से लोड (~40 KB gzipped)
