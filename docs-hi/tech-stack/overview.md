# तकनीकी स्टैक अवलोकन

JanVayu जानबूझकर न्यूनतम स्टैक पर बनाया गया है — शून्य फ्रंटएंड फ्रेमवर्क, तीन npm डिपेंडेंसी, और सर्वरलेस बैकएंड।

---

## स्टैक एक नज़र में

| परत | तकनीक | उद्देश्य |
|-----|--------|---------|
| **फ्रंटएंड** | Vanilla HTML/CSS/JS | एकल-पृष्ठ एप्लिकेशन (कोई बिल्ड स्टेप नहीं) |
| **चार्ट** | Chart.js (CDN) | AQI ट्रेंड विज़ुअलाइज़ेशन |
| **मानचित्र** | Leaflet.js + OpenStreetMap (CDN) | इंटरैक्टिव स्टेशन मानचित्र |
| **बैकएंड** | Netlify Functions (Node.js 18) | सर्वरलेस API एंडपॉइंट |
| **कैश** | Netlify Blobs | स्थायी JSON कैश |
| **ईमेल** | Resend API | दैनिक AQI डाइजेस्ट |
| **AI** | Google Gemini 2.5 Flash | NL प्रश्न, स्वास्थ्य सलाह, विसंगति पहचान |
| **होस्टिंग** | Netlify CDN | GitHub `main` से ऑटो-डिप्लॉय |
| **CI** | GitHub Actions | लिंक जाँच, Dependabot |
| **डेवलपमेंट** | Claude Code (Anthropic) | AI-सहायित डेवलपमेंट |

---

## यह स्टैक क्यों?

### शून्य-फ्रेमवर्क फ्रंटएंड
JanVayu के दर्शकों में 2G कनेक्शन और कम-लागत Android फ़ोन वाले लोग शामिल हैं। React या Vue 40-100 KB JavaScript जोड़ेगा।

### केवल 3 npm डिपेंडेंसी
```json
{
  "@google/generative-ai": "^0.24.1",
  "@netlify/blobs": "^8.1.0",
  "resend": "^6.9.3"
}
```
तीनों केवल सर्वर-साइड हैं। क्लाइंट पर शून्य npm डिपेंडेंसी।

### सर्वरलेस
- शून्य ops बोझ
- निःशुल्क टियर पर्याप्त
- स्वचालित HTTPS और CDN
