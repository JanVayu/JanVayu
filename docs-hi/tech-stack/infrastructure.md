# इन्फ्रास्ट्रक्चर

JanVayu पूरी तरह Netlify प्लेटफ़ॉर्म पर चलता है। कोई पारंपरिक सर्वर, डेटाबेस, या कंटेनर नहीं।

---

## Netlify
- **CDN:** वैश्विक एज नेटवर्क
- **डिप्लॉय:** GitHub `main` से स्वचालित
- **SSL:** Let's Encrypt (स्वचालित)
- **डोमेन:** janvayu.in

## GitHub
- **CI:** GitHub Actions (लिंक जाँच)
- **Dependabot:** मासिक अपडेट
- **Git Hooks:** कमिट संदेश और सुरक्षा जाँच

## लागत: ~₹800/वर्ष
सब कुछ निःशुल्क टियर पर — केवल डोमेन नवीनीकरण की लागत।

विस्तृत जानकारी के लिए [डिप्लॉयमेंट](../technical/deployment.md) पृष्ठ देखें।
