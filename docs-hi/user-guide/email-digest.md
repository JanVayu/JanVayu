# दैनिक ईमेल डाइजेस्ट

JanVayu रोज़ सुबह 8 बजे (IST) एक AQI ईमेल डाइजेस्ट भेजता है।

---

## डाइजेस्ट में क्या होता है

- आपके चुने हुए शहरों का **रियल-टाइम AQI**
- PM2.5 स्तर और **स्वास्थ्य श्रेणी**
- **कल से तुलना** — AQI बढ़ा या घटा
- **स्वास्थ्य सुझाव** — आज के AQI के अनुसार

---

## सदस्यता कैसे लें

1. JanVayu प्लेटफ़ॉर्म पर जाएँ
2. "दैनिक डाइजेस्ट" अनुभाग में अपना ईमेल दर्ज करें
3. अपने शहर चुनें
4. "सदस्यता लें" पर क्लिक करें

---

## तकनीकी विवरण

| पहलू | विवरण |
|------|-------|
| भेजने का समय | 8:00 AM IST (दैनिक) |
| ईमेल सेवा | Resend API |
| भेजने का पता | digest@janvayu.in |
| डेटा स्रोत | WAQI API (लाइव) |
| फ़ंक्शन | `daily-digest.mjs` (Netlify scheduled function) |

---

## सदस्यता रद्द करना

हर ईमेल में "सदस्यता रद्द करें" लिंक होता है। एक क्लिक में रद्द।
