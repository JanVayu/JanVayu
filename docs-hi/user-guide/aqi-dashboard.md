# AQI डैशबोर्ड

लाइव AQI डैशबोर्ड JanVayu की मुख्य सुविधा है। यह WAQI API से रियल-टाइम वायु गुणवत्ता डेटा प्रदर्शित करता है।

---

## AQI स्केल

| AQI रेंज | स्तर | रंग | स्वास्थ्य प्रभाव |
|----------|------|-----|-----------------|
| 0–50 | अच्छा | हरा | कोई चिंता नहीं |
| 51–100 | संतोषजनक | पीला | संवेदनशील लोगों को हल्की परेशानी |
| 101–200 | मध्यम | नारंगी | श्वसन रोगियों को कठिनाई |
| 201–300 | खराब | लाल | सभी को स्वास्थ्य प्रभाव |
| 301–400 | बहुत खराब | बैंगनी | गंभीर स्वास्थ्य चेतावनी |
| 401–500 | गंभीर | मैरून | आपातकालीन स्थिति |

---

## डैशबोर्ड सुविधाएँ

### हीरो सेक्शन
- सबसे खराब शहर का AQI बड़े फ़ॉन्ट में
- GRAP स्तर (दिल्ली-NCR के लिए)
- अंतिम अपडेट समय

### शहर सूची
- 40+ शहरों का PM2.5 और AQI
- रंग-कोडित कार्ड
- सबसे खराब → सबसे अच्छा क्रम

### इंटरैक्टिव मानचित्र
- Leaflet.js + OpenStreetMap
- हर स्टेशन पर क्लिक करके विवरण देखें
- रंग-कोडित मार्कर

### चार्ट
- मेट्रो बनाम क्षेत्रीय तुलना (Chart.js)
- PM2.5 ट्रेंड लाइन

---

## डेटा स्रोत

- **WAQI API** — 500+ स्टेशनों से रियल-टाइम डेटा
- **CPCB CAAQMS** — आधिकारिक भारतीय AQI डेटा
- हर 10 मिनट में ऑटो-रिफ्रेश
