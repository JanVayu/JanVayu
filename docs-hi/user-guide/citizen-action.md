# नागरिक कार्रवाई उपकरण

JanVayu नागरिकों को वायु प्रदूषण के खिलाफ़ कार्रवाई करने के लिए उपकरण प्रदान करता है।

---

## उपलब्ध उपकरण

### 1. RTI (सूचना का अधिकार) मार्गदर्शिका
- RTI आवेदन कैसे दाखिल करें
- प्रासंगिक विभाग (CPCB, SPCB, MCD, DPCC)
- नमूना RTI प्रश्न
- ऑनलाइन RTI पोर्टल लिंक

### 2. शिकायत दर्ज करें
- CPCB शिकायत पोर्टल
- MCD हेल्पलाइन नंबर
- DPCC ऑनलाइन शिकायत
- सम्पर्क प्रशासन पोर्टल

### 3. AI जवाबदेही ब्रीफ़
- अपने शहर का नाम दर्ज करें
- AI एक संरचित ब्रीफ़ तैयार करेगा:
  - वर्तमान AQI स्थिति
  - NCAP लक्ष्य बनाम वास्तविकता
  - स्थानीय प्रदूषण स्रोत
  - 5 जवाबदेही प्रश्न
- वार्ड पार्षद, पत्रकार, या RWA को दें

### 4. नागरिक गवाहियाँ
- लोगों के अनुभव पढ़ें
- अपनी गवाही साझा करें
- स्वास्थ्य प्रभावों का दस्तावेज़ीकरण

---

## कार्रवाई कैसे करें

### व्यक्तिगत स्तर पर
1. AQI ऐप इंस्टॉल करें / JanVayu बुकमार्क करें
2. उच्च AQI दिनों पर मास्क पहनें
3. एयर प्यूरीफायर का उपयोग करें
4. स्वास्थ्य डाइजेस्ट की सदस्यता लें

### सामुदायिक स्तर पर
1. RWA मीटिंग में AQI डेटा प्रस्तुत करें
2. वार्ड पार्षद को जवाबदेही ब्रीफ़ दें
3. स्थानीय मीडिया को डेटा साझा करें
4. RTI दाखिल करें

### नीति स्तर पर
1. NCAP बजट उपयोग की निगरानी करें
2. सुप्रीम कोर्ट आदेशों के अनुपालन की जाँच करें
3. स्थानीय निकाय की बैठकों में भाग लें
4. जनप्रतिनिधियों से प्रश्न पूछें
