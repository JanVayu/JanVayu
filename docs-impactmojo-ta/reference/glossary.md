# சொற்களஞ்சியம்

ImpactMojo இன் தளம் மற்றும் ஆவணங்களில் பயன்படுத்தப்படும் முக்கிய சொற்கள்.

---

## தள சொற்கள்

| சொல் | வரையறை |
|------|-----------|
| **முன்னணி பாடநெறி** | ~13 தொகுதிகள், சொற்களஞ்சியம், வழக்கு ஆய்வுகள் மற்றும் சிந்தனை தூண்டல்கள் கொண்ட ஆழமான நிகழ்ச்சி |
| **அடிப்படை பாடநெறி** | குறுகிய, சுயவேகக் கற்றல் திறன்-மைய பாடநெறி (30–60 நிமிடங்கள்) |
| **Course Deck** | இந்திய நாட்டுப்புற கலை ஓவியங்களுடன் காட்சி 101 ஸ்லைடு வழங்கல் (23 கிடைக்கின்றன) |
| **ஊடாடும் விளையாட்டு** | உண்மையான அளவுருக்களைப் பயன்படுத்தும் உலாவி-அடிப்படையிலான பொருளாதார உருவகப்படுத்தல், விவாதத்திற்காக வடிவமைக்கப்பட்டது |
| **ஊடாடும் ஆய்வகம்** | உறுதியான வெளியீடுகளை உருவாக்கும் நடைமுறை பணிமேடை (ToC வரைபடம், ஆராய்ச்சிக் கேள்வி போன்றவை) |
| **கையேடு** | ஒரு பக்கக் குறிப்புப் பொருள் — சுருக்கத் தாள் அல்லது சுருக்கக் குறிப்பு |
| **ImpactLex** | PWA ஆகக் கிடைக்கும் 500+ வளர்ச்சி சொற்களின் தேடக்கூடிய சொற்களஞ்சியம் |
| **வளர்ச்சி வழக்கு ஆய்வு** | 117 நாடுகளிலிருந்து ஆவணப்படுத்தப்பட்ட நிஜ-உலக திட்டம் |
| **DevDiscourse** | நிபுணர் பொருத்தத்திற்காகத் தேர்ந்தெடுக்கப்பட்ட தொகுக்கப்பட்ட கல்விக் கட்டுரை அல்லது புத்தகம் |
| **Dataverse** | 247 வளர்ச்சி தரவுத்தொகுப்புகள், API-கள், கருவிகள் மற்றும் MCP சேவையகங்களின் சேகரிப்பு |
| **BookSummary** | அத்தியாய வழிசெலுத்தல், ஆதார ஆய்வு மற்றும் AI கேள்வி-பதில் கொண்ட புத்தகத்திற்கான ஊடாடும் துணை |
| **கற்றல் பாதை** | பங்கு/ஆர்வத்தின் அடிப்படையில் ஒழுங்கமைக்கப்பட்ட உள்ளடக்கம் (எ.கா., MEL & ஆராய்ச்சி, பொருளாதாரம் & கொள்கை) |
| **கற்றல் வழி** | குறிப்பிட்ட கற்றல் இலக்குக்கான தொகுக்கப்பட்ட பாடநெறிகள், விளையாட்டுகள் மற்றும் ஆய்வகங்களின் வரிசை |
| **கற்றல் தொடர்ச்சி** | பயனர் தளத்தில் ஈடுபடும் தொடர்ச்சியான நாட்களின் எண்ணிக்கை |

## வளர்ச்சி & ஆராய்ச்சி சொற்கள்

| சொல் | வரையறை |
|------|-----------|
| **MEL** | Monitoring, Evaluation, and Learning (கண்காணிப்பு, மதிப்பீடு மற்றும் கற்றல்) |
| **ToC** | Theory of Change — செயல்பாடுகள் எவ்வாறு நோக்கப்பட்ட விளைவுகளுக்கு இட்டுச்செல்கின்றன என்பதை வரைபடமாக்கும் மாதிரி |
| **RCT** | Randomised Controlled Trial — தாக்க மதிப்பீட்டிற்கான சோதனை வடிவமைப்பு |
| **Logframe** | Logical Framework — செயல்பாடுகள், வெளியீடுகள், விளைவுகள் மற்றும் தாக்கத்தை இணைக்கும் திட்டமிடல் கருவி |
| **RQ** | Research Question — ஒரு ஆய்வு பதிலளிக்க நோக்கும் மைய கேள்வி |
| **DHS** | Demographic and Health Surveys — பல நாட்டு வீட்டு கணக்கெடுப்பு நிகழ்ச்சி |
| **NFHS** | National Family Health Survey — இந்தியாவின் DHS சமமானது |
| **NSSO** | National Sample Survey Office — இந்தியாவின் தேசிய புள்ளியியல் கணக்கெடுப்பு அமைப்பு |
| **MCP** | Model Context Protocol — AI கருவிகளை தரவு ஆதாரங்களுடன் இணைப்பதற்கான தரம் |
| **PWA** | Progressive Web App — ஆஃப்லைன் ஆதரவுடன் சாதனங்களில் நிறுவக்கூடிய வலை பயன்பாடு |

## தொழில்நுட்ப சொற்கள்

| சொல் | வரையறை |
|------|-----------|
| **Edge Function** | CDN விளிம்பில் இயங்கும் சேவையக-பக்க செயல்பாடு (Netlify/Supabase) |
| **JWT** | JSON Web Token — நிலையற்ற அங்கீகாரத்திற்கான கையொப்பமிடப்பட்ட டோக்கன் |
| **RLS** | Row-Level Security — பயனருக்கான தரவு அணுகலை கட்டுப்படுத்தும் Supabase/PostgreSQL அம்சம் |
| **Auth Gate** | அணுகல் டோக்கன்களை சரிபார்க்கும் பிரீமியம் கருவி தளங்களில் உள்ள Edge Function |
| **Anon Key** | கிளையன்ட்-பக்க பயன்பாட்டிற்கு பாதுகாப்பான Supabase பொது விசை (RLS ஐ அமல்படுத்துகிறது) |
| **Service Role Key** | RLS ஐ புறக்கணிக்கும் Supabase நிர்வாக விசை (சேவையக-பக்கம் மட்டும்) |
