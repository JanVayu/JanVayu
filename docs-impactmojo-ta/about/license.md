# உரிமம்

ImpactMojo திறந்த தன்மை மற்றும் உள்ளடக்கப் பாதுகாப்பை சமநிலைப்படுத்த இரட்டை உரிமத்தைப் பயன்படுத்துகிறது.

## குறியீடு — MIT உரிமம்

அனைத்து மூலக் குறியீடும் (HTML, CSS, JavaScript, கட்டமைப்புக் கோப்புகள், Netlify/Supabase செயல்பாடுகள்) [MIT உரிமத்தின்](https://opensource.org/licenses/MIT) கீழ் உரிமம் பெற்றது.

நீங்கள் சுதந்திரமாக:
- பிரதிகளை பயன்படுத்தலாம், நகலெடுக்கலாம், மாற்றலாம், இணைக்கலாம், வெளியிடலாம், விநியோகிக்கலாம், துணை உரிமம் வழங்கலாம் மற்றும் விற்கலாம்
- வணிக மற்றும் வணிக அல்லாத திட்டங்களில் பயன்படுத்தலாம்

ஒரே தேவை அசல் பதிப்புரிமை அறிவிப்பு மற்றும் உரிமத்தை உள்ளடக்குவதாகும்.
