# ImpactMojo ஆவணங்கள்

**ImpactMojo** என்பது தென்னாசியா முழுவதும் சர்வதேச வளர்ச்சி, சமூக தாக்கம் மற்றும் பொதுக் கொள்கை துறைகளில் பணிபுரிபவர்களுக்கான இலவச கற்றல் தளமாகும். இது பாடநெறிகள், விளையாட்டுகள், ஆய்வகங்கள், கையேடுகள், வழக்கு ஆய்வுகள் மற்றும் தரவு கருவிகள் — அனைத்தையும் பணச்சுவர் அல்லது உள்நுழைவு தேவையின்றி வழங்குகிறது.

> **நேரடி தளம்:** [www.impactmojo.in](https://www.impactmojo.in)
> **GitHub:** [github.com/JanVayu/ImpactMojo](https://github.com/JanVayu/ImpactMojo)

---

## ImpactMojo என்றால் என்ன?

தென்னாசியா முழுவதும் உள்ள வளர்ச்சி நிபுணர்கள், ஆராய்ச்சியாளர்கள், தன்னார்வ நிறுவன ஊழியர்கள் மற்றும் மாணவர்களுக்கு MEL, ஆராய்ச்சி முறைகள், பொருளாதாரம் மற்றும் கொள்கை ஆகியவற்றில் அணுகக்கூடிய, உயர்தரக் கல்வி தேவை. ImpactMojo 48 பாடநெறிகள், 16 விளையாட்டுகள், 19 ஆய்வகங்கள், 400+ கையேடுகள், 200 வழக்கு ஆய்வுகள், 500+ தொகுக்கப்பட்ட கட்டுரைகள் மற்றும் 247 தரவுத்தொகுப்புகள் — அனைத்தும் பயன்பாட்டின் போது இலவசமாக வழங்குகிறது.

## வழிகாட்டும் கொள்கைகள்

- **பயன்பாட்டின் போது இலவசம்** — முக்கிய உள்ளடக்கத்திற்கு உள்நுழைவு அல்லது கட்டணம் தேவையில்லை
- **நிபுணர்களுக்கு முன்னுரிமை** — படிப்பவர்களுக்கு மட்டுமல்ல, பணிபுரிபவர்களுக்காக வடிவமைக்கப்பட்டது
- **தென்னாசியா மையம்** — இப்பகுதியின் வழக்கு ஆய்வுகள், தரவு மற்றும் சூழல்
- **அணுகக்கூடியது** — மெதுவான இணைப்புகள், பழைய சாதனங்கள் மற்றும் PWA வழியாக ஆஃப்லைனில் வேலை செய்கிறது
