# Dataverse வழிகாட்டி

ImpactMojo Dataverse என்பது World Bank, DHS, NFHS மற்றும் NSSO உள்ளிட்ட முக்கிய ஆதாரங்களிலிருந்து 247 வளர்ச்சி தரவுத்தொகுப்புகள், API-கள், கருவிகள் மற்றும் MCP சேவையகங்களின் தொகுக்கப்பட்ட சேகரிப்பாகும்.

---

## Dataverse என்றால் என்ன?

Dataverse நிபுணர்கள், ஆராய்ச்சியாளர்கள் மற்றும் மாணவர்களுக்கு பல தனி போர்ட்டல்களை வழிசெலுத்தாமல் வளர்ச்சி தரவைக் கண்டறிந்து பயன்படுத்த உதவுகிறது. ஒவ்வொரு பதிவும் பின்வருவனவற்றை உள்ளடக்குகிறது:
- **விவரம்** — தரவுத்தொகுப்பு எதை உள்ளடக்குகிறது
- **ஆதாரம்** — தோற்றுவிக்கும் நிறுவனம்
- **அணுகல் முறை** — நேரடி பதிவிறக்கம், API அல்லது MCP சேவையகம்
- **குறிச்சொற்கள்** — நாடு, கருப்பொருள், முறையியல், அலைவரிசை

## தரவு ஆதாரங்கள்

| ஆதாரம் | வகை | பரப்பு |
|--------|------|---------|
| World Bank Open Data | API + பதிவிறக்கம் | உலகளாவிய, 200+ நாடுகள் |
| DHS (Demographic and Health Surveys) | பதிவிறக்கம் | 90+ நாடுகள் |
| NFHS (National Family Health Survey) | பதிவிறக்கம் | இந்தியா, மாநில அளவு |
| NSSO (National Sample Survey Office) | பதிவிறக்கம் | இந்தியா, தேசிய/மாநில |
| UDISE+ (பள்ளிக் கல்வி) | பதிவிறக்கம் | இந்தியா, மாவட்ட அளவு |
| ASER (ஆண்டு கல்வி நிலை அறிக்கை) | பதிவிறக்கம் | இந்தியா, கிராமப்புற மாவட்டங்கள் |
| RBI (இந்திய ரிசர்வ் வங்கி) | API + பதிவிறக்கம் | இந்தியா, பொருளாதார |
| இந்திய மக்கள்தொகை கணக்கெடுப்பு | பதிவிறக்கம் | இந்தியா, கிராமம்/வார்டு அளவு |

## Dataverse ஐப் பயன்படுத்துதல்

### கருப்பொருள் மூலம் உலாவுங்கள்

தரவுத்தொகுப்புகள் பின்வரும் கருப்பொருள்களில் குறிச்சொல்லிடப்பட்டுள்ளன:
- சுகாதாரம் & ஊட்டச்சத்து
- கல்வி
- வறுமை & சமத்துவமின்மை
- ஆட்சி & நிறுவனங்கள்
- சூழல் & காலநிலை
- பாலினம் & சமூக உள்ளடக்கம்
- பொருளாதார வளர்ச்சி
- வேளாண்மை & கிராமப்புறம்

### தேடல்

முக்கிய வார்த்தை, ஆதாரம் அல்லது நாடு மூலம் தரவுத்தொகுப்புகளைக் கண்டறிய தேடல் பட்டியை அல்லது **Ctrl+K** ஐப் பயன்படுத்துங்கள்.

### MCP சேவையகங்கள்

தேர்ந்தெடுக்கப்பட்ட தரவுத்தொகுப்புகள் AI கருவிகள் மற்றும் தானியங்கு பணிப்போக்குகளுடன் ஒருங்கிணைப்புக்காக MCP (Model Context Protocol) சேவையகங்களாகக் கிடைக்கின்றன.
