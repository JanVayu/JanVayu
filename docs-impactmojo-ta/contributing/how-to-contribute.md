# பங்களிப்பது எப்படி

ImpactMojo திறந்த மூலமாகும். குறியீடு, உள்ளடக்கம், மொழிபெயர்ப்புகள் மற்றும் யோசனைகளின் பங்களிப்புகளை நாங்கள் வரவேற்கிறோம்.

---

## பங்களிக்கும் வழிகள்

| வகை | எப்படி |
|------|-----|
| **பிழை புகாரளிப்பு** | [GitHub Issue](https://github.com/Varnasr/ImpactMojo/issues) திறக்கவும் |
| **அம்ச கோரிக்கைகள்** | [GitHub Discussion](https://github.com/Varnasr/ImpactMojo/discussions) தொடங்கவும் |
| **குறியீடு பங்களிப்புகள்** | Fork → branch → PR (கீழே உள்ள பணிப்போக்கைப் பார்க்கவும்) |
| **உள்ளடக்க பங்களிப்புகள்** | பாடநெறிகள், வழக்கு ஆய்வுகள், கையேடுகள், DevDiscourses |
| **மொழிபெயர்ப்புகள்** | [மொழிபெயர்ப்புகள்](translations.md) பார்க்கவும் |
| **தரவு பங்களிப்புகள்** | Dataverse க்கான தரவுத்தொகுப்புகள் |

## குறியீடு பங்களிப்பு பணிப்போக்கு

```bash
# 1. GitHub இல் repo ஐ Fork செய்யுங்கள்

# 2. உங்கள் fork ஐ Clone செய்யுங்கள்
git clone https://github.com/<your-username>/ImpactMojo.git
cd ImpactMojo

# 3. ஒரு கிளையை உருவாக்குங்கள்
git checkout -b feature/your-feature-name

# 4. மாற்றங்கள் செய்யுங்கள் (உருவாக்க படி தேவையில்லை — HTML/CSS/JS ஐ நேரடியாக திருத்தவும்)

# 5. உள்ளூரில் சோதிக்கவும்
# உலாவியில் index.html ஐ திறக்கவும், அல்லது பயன்படுத்தவும்:
npx serve .

# 6. தெளிவான செய்தியுடன் commit செய்யுங்கள்
git commit -m "Add: description of your change"

# 7. Push செய்து PR திறக்கவும்
git push origin feature/your-feature-name
```

### Commit செய்தி மரபு

| முன்னொட்டு | பயன்பாடு |
|--------|---------|
| `Add` | புதிய அம்சம் அல்லது உள்ளடக்கம் |
| `Fix` | பிழை திருத்தம் |
| `Update` | ஏற்கனவே உள்ள அம்சத்திற்கான மேம்பாடு |
| `Translate` | மொழிபெயர்ப்பு சேர்ப்புகள் அல்லது மேம்பாடுகள் |
| `Docs` | ஆவண மாற்றங்கள் |
| `Refactor` | நடத்தை மாற்றமின்றி குறியீடு மறுகட்டமைப்பு |
| `Chore` | பராமரிப்பு, சார்புகள், CI |

## உள்ளடக்க பங்களிப்புகள்

### பாடநெறிகள்
- முன்னணி அல்லது அடிப்படை பாடநெறி வார்ப்புருவைப் பின்பற்றுங்கள்
- முக்கிய சொற்களின் சொற்களஞ்சியத்தைச் சேர்க்கவும்
- தொடர்புடைய இடங்களில் தென்னாசிய வழக்கு ஆய்வுகளைச் சேர்க்கவும்
- எளிய மொழியில் எழுதுங்கள் — இலக்கு வாசிப்பு நிலை இளங்கலை

### வழக்கு ஆய்வுகள்
- எந்த நாட்டிலிருந்தும் நிஜ-உலக திட்டங்கள்
- நாடு, கருப்பொருள் மற்றும் முறையியல் மூலம் குறிச்சொல்லிடவும்
- ஆதாரங்கள் மற்றும் மேற்கோள்களைச் சேர்க்கவும்

### கையேடுகள்
- ஒரு பக்கத்திற்கு ஒரு கருத்து
- எளிய மொழி, குறைந்தபட்ச தொழில்நுட்பச் சொற்கள்
- சாத்தியமான இடங்களில் ஒரு காட்சி (வரைபடம், அட்டவணை அல்லது ஓட்ட வரைபடம்) சேர்க்கவும்
- முதன்மை ஆதாரங்களை மேற்கோள் காட்டவும்

### Dataverse பதிவுகள்
- சேர்க்கவும்: தரவுத்தொகுப்பு பெயர், ஆதாரம், அணுகல் முறை (பதிவிறக்கம்/API/MCP), குறிச்சொற்கள்
- தெளிவான உரிமத்துடன் திறந்த-அணுகல் தரவுத்தொகுப்புகளை விரும்புங்கள்
