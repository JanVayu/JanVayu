# স্কিলস ও AI প্রম্পটস

এই অনুচ্ছেদ AI স্কিল ফাইল — সিস্টেম প্রম্পট, নির্দেশনা প্যাটার্ন, এবং প্রম্পট ইঞ্জিনিয়ারিং সিদ্ধান্তসমূহ — নথিভুক্ত করে।

---

## স্কিল ফাইল কী?

একটি কাঠামোবদ্ধ সিস্টেম প্রম্পট যা AI মডেলকে বলে একটি নির্দিষ্ট প্রসঙ্গে কীভাবে আচরণ করতে হবে:
- মডেলের **ভূমিকা ও ব্যক্তিত্ব**
- প্রাপ্ত **ডেটা**
- **আউটপুট ফরম্যাট**
- **সুর, দৈর্ঘ্য, এবং দাবির উপর সীমাবদ্ধতা**
- এড়ানো উচিত এমন **ব্যর্থতার ধরন**

## সাধারণ প্রম্পটিং নীতি

1. **বাস্তব ডেটার উপর ভিত্তি করে** — মডেল কখনো স্মৃতি থেকে উত্তর দেয় না
2. **সুনির্দিষ্ট আউটপুট ফরম্যাট** — বিশেষত জবাবদিহিতা ব্রিফের জন্য
3. **দৈর্ঘ্য সীমিত** — প্রতিটি প্রম্পটে স্পষ্ট শব্দ/টোকেন সীমা
4. **ব্যর্থতার ধরন নামকরণ** — "সাধারণ পরামর্শ দেবেন না"
5. **বহুভাষিক** — "প্রশ্নের ভাষায় উত্তর দিন"
6. **গ্রেসফুল ফলব্যাক** — প্রতিটি ফাংশনে non-AI বিকল্প
