# স্কিল: অটোমেশন

## নির্ধারিত কাজ

### ফিড রিফ্রেশ (প্রতি 4 ঘণ্টায়)
- **ফাংশন:** `scheduled-fetch.mjs`
- Reddit, Twitter/X, Instagram, সংবাদ ফেচ
- Netlify Blobs-এ ক্যাশ
- ব্যর্থতায় পুরোনো ক্যাশ বজায় রাখে

### দৈনিক ডাইজেস্ট (সকাল 8টা IST)
- **ফাংশন:** `daily-digest.mjs`
- সদস্যদের শহরের AQI ফেচ
- HTML ইমেল প্রস্তুত
- Resend API দিয়ে পাঠানো

## স্বয়ংক্রিয় পরীক্ষা

### অসঙ্গতি শনাক্তকরণ
- পেজ লোডে ট্রিগার
- 5টি শহরের মৌসুমী বেসলাইনের সাথে তুলনা
- 10-মিনিট ক্যাশ

### লিঙ্ক পরীক্ষা (CI)
- GitHub Actions (push/PR-এ)
- Lychee লিঙ্ক চেকার
- সোশ্যাল মিডিয়া ও localhost URL বাদ দেয়

### Dependabot
- মাসিক npm আপডেট পরীক্ষা
- GitHub Actions আপডেট পরীক্ষা
