# স্কিল: ভিজ্যুয়াল ডিজাইন

JanVayu-র UI/UX সিদ্ধান্ত ও ডিজাইন নীতি।

---

## ডিজাইন নীতি

1. **কার্যকারিতা প্রথমে** — সৌন্দর্যের আগে ব্যবহারযোগ্যতা
2. **মোবাইল-ফার্স্ট** — অধিকাংশ ভারতীয় ব্যবহারকারী মোবাইলে
3. **কম ব্যান্ডউইথ** — ন্যূনতম ছবি, কোনো ভিডিও অটোপ্লে নেই
4. **সুলভ** — WCAG AA অনুসরণ

---
