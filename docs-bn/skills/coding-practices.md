## প্রম্পট প্যাটার্ন

### 1. "নাগরিক ডেটা" প্যাটার্ন
```
একটি [ফিচার] লিখুন। ডেটা [উৎস] থেকে আসে।
Plain HTML/CSS/JS-এ, কোনো ফ্রেমওয়ার্ক নেই।
দর্শক 2G কানেকশনে থাকতে পারে।
লোড গতিকে অগ্রাধিকার দিন।
```

### 2. "সার্ভারলেস ফাংশন" প্যাটার্ন
```
Netlify Function (ES module, .mjs) লিখুন:
- CORS প্রিফ্লাইট হ্যান্ডেল করুন
- JSON রিটার্ন করুন
- try/catch প্রতিটি বাহ্যিক কলে
- সিক্রেট process.env থেকে পড়ুন
- গ্রেসফুল ফলব্যাক অন্তর্ভুক্ত করুন
```

### 3. "ডিবাগিং" প্যাটার্ন
```
এই ফাংশন [ত্রুটি] রিটার্ন করছে।
ফাংশন আবার লিখবেন না।
সমস্যার নির্দিষ্ট লাইন চিহ্নিত করুন।
ন্যূনতম পরিবর্তন প্রস্তাব করুন।
```
