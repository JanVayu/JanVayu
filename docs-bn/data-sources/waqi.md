# রিয়েল-টাইম AQI (WAQI)

WAQI (World Air Quality Index) API হল JanVayu-র প্রাথমিক রিয়েল-টাইম ডেটা উৎস।

---

## WAQI কী?

WAQI প্রজেক্ট সারা বিশ্বের 12,000+ মনিটরিং স্টেশন থেকে রিয়েল-টাইম বায়ুমান ডেটা সংগ্রহ করে।

---

## JanVayu-তে ব্যবহার

- **500+ ভারতীয় স্টেশন** কভার
- **প্রতি 10 মিনিটে** অটো-রিফ্রেশ
- **ক্লায়েন্ট-সাইড** কল (সরাসরি ব্রাউজার থেকে)
- **ফ্রি-টায়ার** পাবলিক টোকেন

## উপলব্ধ ডেটা
- AQI (Air Quality Index)
- PM2.5 (µg/m³)
- PM10, NO2, SO2, CO, O3
- তাপমাত্রা, আর্দ্রতা, বাতাসের গতি

## API এন্ডপয়েন্ট
- `api.waqi.info/feed/{city}/` — একক শহর
- `api.waqi.info/map/bounds/` — ভৌগোলিক সীমানার মধ্যে স্টেশন
