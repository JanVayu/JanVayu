# শেয়ারিং ও রেপ্লিকেশন

JanVayu-র Claude Code ওয়ার্কফ্লো কীভাবে শেয়ার করবেন বা আপনার প্রজেক্টের জন্য পুনরাবৃত্তি করবেন।

---

## টিমের জন্য

### নতুন অবদানকারীকে যুক্ত করুন
1. রিপো ক্লোন করুন
2. এই GitBook ডকুমেন্টেশন পড়ুন
3. Claude Code ইনস্টল করুন: `npm install -g @anthropic-ai/claude-code`
4. সেশন শুরু করুন: `cd JanVayu && claude`

---

## আপনার প্রজেক্টের জন্য

### ধাপ 1: সীমাবদ্ধতা-প্রথম পদ্ধতি অবলম্বন করুন
- নির্ধারণ করুন প্রজেক্ট কী *ব্যবহার করবে না*
- `CLAUDE.md` ফাইলে লিখুন

### ধাপ 2: স্কিল ফাইল তৈরি করুন
AI সুবিধাগুলোর জন্য স্ট্রাকচার্ড সিস্টেম প্রম্পট নথিভুক্ত করুন।

### ধাপ 3: Git Hooks সেটআপ করুন
JanVayu-র `.githooks/` কপি করুন।

---

## JanVayu অন্য শহরের জন্য ফর্ক করা

JanVayu MIT-লাইসেন্সড:
1. GitHub-এ ফর্ক করুন
2. `index.html`-এ শহরের তালিকা আপডেট করুন
3. `anomaly-check.mjs`-এ মৌসুমী বেসলাইন পরিবর্তন করুন
4. WAQI স্টেশন ID আপডেট করুন
5. স্থানীয় ভাষায় অনুবাদ করুন
6. আপনার নিজের Netlify সাইট সেটআপ করুন

Claude Code এই সমস্ত ধাপে সাহায্য করতে পারে।
