# Claude Code — JanVayu কীভাবে তৈরি হয়েছে

JanVayu-র উন্নয়ন **Claude Code** — Anthropic-এর অফিসিয়াল CLI এজেন্ট — এর সহায়তায় হয়েছে।

---

## Claude Code কী?

Claude Code একটি টার্মিনাল-ভিত্তিক AI কোডিং এজেন্ট। আপনি এটি আপনার প্রজেক্ট ডিরেক্টরিতে চালান এবং এটি:
- ফাইল পড়তে ও লিখতে পারে
- শেল কমান্ড চালাতে পারে
- কোডবেস খুঁজতে পারে
- কমিট ও পুল রিকুয়েস্ট তৈরি করতে পারে
- জটিল মাল্টি-ফাইল রিফ্যাক্টর সামলাতে পারে

---

## JanVayu-র জন্য Claude Code-এর ব্যবহার

### প্রাথমিক নির্মাণ
- `index.html`-এ সম্পূর্ণ frontend
- সমস্ত 13টি Netlify Functions
- ইনফ্রাস্ট্রাকচার কনফিগ

### v25.1 AI সুবিধাসমূহ
- চারটি Gemini-চালিত সুবিধাই Claude Code সেশনে তৈরি
- Gemini সিস্টেম প্রম্পট প্রস্তুত করা হয়েছে
- CHANGELOG ও PR পরিচালনা করা হয়েছে

### ডকুমেন্টেশন
- এই সম্পূর্ণ GitBook ডকুমেন্টেশন Claude Code তৈরি করেছে — এই পৃষ্ঠা সহ

## মূল শক্তিসমূহ

1. **সম্পূর্ণ কোডবেস প্রসঙ্গ** — Claude আপনার কোড পড়ে লেখার আগে
2. **সীমাবদ্ধতা-প্রথম** — "কোনো ফ্রেমওয়ার্ক নেই" এর মতো সীমাবদ্ধতা বোঝে
3. **মাল্টি-ফাইল সমন্বয়** — একটি প্রম্পটে Functions, HTML, CSS, JS সব পরিবর্তন
4. **Git-নেটিভ** — ব্রাঞ্চ, কমিট, PR একই টুলে
