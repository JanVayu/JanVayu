## সাধারণ নিয়ম

- **Vanilla JavaScript** — কোনো ফ্রেমওয়ার্ক নেই
- **ES2020** — নতুন ফিচার নয় (ব্রাউজার সামঞ্জস্যতা)
- **কোনো বিল্ড স্টেপ নেই**
- **ক্লায়েন্টে কোনো npm ডিপেন্ডেন্সি নেই**

---

## HTML
- সিমান্টিক, সুলভ মার্কআপ
- ARIA roles যেখানে প্রয়োজন
- সমস্ত ইমেজে alt text

## CSS
- CSS Custom Properties থিমিংয়ের জন্য
- মোবাইল-ফার্স্ট রেসপন্সিভ
- WCAG AA কালার কন্ট্রাস্ট

## JavaScript
- Fetch API (axios নয়)
- DOM API (jQuery নয়)
- try/catch প্রতিটি বাহ্যিক কলে
- console.log ডিবাগিংয়ের জন্য (throw নয়)

## Netlify Functions
- CORS প্রিফ্লাইট হ্যান্ডেল করুন
- JSON রিটার্ন করুন
- process.env থেকে সিক্রেট পড়ুন
- গ্রেসফুল ফলব্যাক আবশ্যক
