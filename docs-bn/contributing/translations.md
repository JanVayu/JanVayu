## কীভাবে অনুবাদ করবেন

### প্ল্যাটফর্ম (index.html)
ভাষার স্ট্রিংগুলো `index.html`-এ JS অবজেক্ট হিসেবে আছে। নতুন ভাষা যোগ করতে:
1. বিদ্যমান ভাষা অবজেক্ট কপি করুন
2. সমস্ত স্ট্রিং অনুবাদ করুন
3. ভাষা টগলে অপশন যোগ করুন

### ডকুমেন্টেশন (Docsify)
এই ডকুমেন্টেশন `docs-bn/` ডিরেক্টরিতে বাংলা অনুবাদ সহ আছে। অন্যান্য ভাষার জন্য:
- `docs-hi/` — হিন্দি
- `docs-ta/` — তামিল
- `docs-mr/` — মারাঠি

## অনুবাদ নির্দেশিকা

1. প্রযুক্তিগত শব্দ ইংরেজিতে রাখুন (AQI, PM2.5, API, ইত্যাদি)
2. সহজ, চলিত ভাষা ব্যবহার করুন
3. সাধু ভাষা বা অতিরিক্ত তৎসম শব্দ এড়িয়ে চলুন
4. সংক্ষিপ্ত থাকুন — দীর্ঘ বাক্য তৈরি করবেন না
