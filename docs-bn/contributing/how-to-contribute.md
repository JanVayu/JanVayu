# কীভাবে অবদান রাখবেন

JanVayu একটি ওপেন সোর্স প্রজেক্ট। আপনার অবদান স্বাগত!

---

## অবদানের উপায়

### 1. কোড অবদান
- বাগ ফিক্স
- নতুন ফিচার
- পারফরম্যান্স উন্নতি
- অ্যাক্সেসিবিলিটি উন্নতি

### 2. কন্টেন্ট অবদান
- ডেটা সোর্স যোগ করা
- নীতি আপডেট
- গবেষণা রেফারেন্স

### 3. অনুবাদ
- বিদ্যমান ভাষায় উন্নতি
- নতুন ভাষা যোগ করা

### 4. বাগ রিপোর্ট
- GitHub Issues-এ রিপোর্ট করুন
- Issue template ব্যবহার করুন

---

## প্রক্রিয়া

1. রিপো ফোর্ক করুন
2. ফিচার ব্রাঞ্চ তৈরি করুন (`git checkout -b feature/my-feature`)
3. পরিবর্তন করুন
4. `netlify dev` দিয়ে লোকালি টেস্ট করুন
5. কমিট করুন (কনভেনশন মানুন)
6. পুল রিকুয়েস্ট তৈরি করুন

---

## কমিট মেসেজ কনভেনশন

প্রতিটি কমিট এই উপসর্গগুলো দিয়ে শুরু হতে হবে:

| উপসর্গ | ব্যবহার |
|--------|---------|
| `Add` | নতুন ফিচার |
| `Fix` | বাগ ফিক্স |
| `Update` | বিদ্যমান ফিচারে উন্নতি |
| `Translate` | অনুবাদ |
| `Docs` | ডকুমেন্টেশন |
| `Refactor` | কোড পুনর্গঠন |

---

## লোকাল সেটআপ

```bash
git clone https://github.com/YOUR-USERNAME/JanVayu.git
cd JanVayu
npm install
netlify dev
```
