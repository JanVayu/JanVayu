# ইনফ্রাস্ট্রাকচার

JanVayu সম্পূর্ণভাবে Netlify প্ল্যাটফর্মে চলে। কোনো প্রচলিত সার্ভার, ডেটাবেস, বা কন্টেইনার নেই।

---

## Netlify
- **CDN:** বৈশ্বিক এজ নেটওয়ার্ক
- **ডিপ্লয়:** GitHub `main` থেকে স্বয়ংক্রিয়
- **SSL:** Let's Encrypt (স্বয়ংক্রিয়)
- **ডোমেইন:** janvayu.in

## GitHub
- **CI:** GitHub Actions (লিঙ্ক যাচাই)
- **Dependabot:** মাসিক আপডেট
- **Git Hooks:** কমিট মেসেজ ও সুরক্ষা যাচাই

সবকিছু ফ্রি টায়ারে — শুধুমাত্র ডোমেইন নবায়নের খরচ।
