# শব্দকোষ

ImpactMojo-র প্ল্যাটফর্ম এবং ডকুমেন্টেশন জুড়ে ব্যবহৃত মূল শব্দাবলী।

---

## প্ল্যাটফর্ম শব্দাবলী

| শব্দ | সংজ্ঞা |
|------|-----------|
| **ফ্ল্যাগশিপ কোর্স** | ~১৩টি মডিউল, অভিধান, কেস স্টাডি এবং প্রতিফলন প্রম্পট সহ গভীর প্রোগ্রাম |
| **ভিত্তিমূলক কোর্স** | সংক্ষিপ্ত, স্ব-গতিসম্পন্ন দক্ষতা-কেন্দ্রিক কোর্স (৩০-৬০ মিনিট) |
| **কোর্স ডেক** | ভারতীয় লোকশিল্প চিত্রসহ ভিজ্যুয়াল 101 স্লাইড প্রেজেন্টেশন (২৩টি উপলব্ধ) |
| **ইন্টারেক্টিভ গেম** | প্রকৃত প্যারামিটার ব্যবহার করে ব্রাউজার-ভিত্তিক অর্থনৈতিক সিমুলেশন, ডিব্রিফের জন্য ডিজাইন করা |
| **ইন্টারেক্টিভ ল্যাব** | বাস্তব আউটপুট তৈরিকারী হাতে-কলমে ওয়ার্কবেঞ্চ (ToC ডায়াগ্রাম, গবেষণা প্রশ্ন ইত্যাদি) |
| **হ্যান্ডআউট** | একক-পৃষ্ঠার রেফারেন্স উপকরণ — চিট শিট বা ব্রিফিং নোট |
| **ImpactLex** | ৫০০+ উন্নয়ন শব্দের অনুসন্ধানযোগ্য শব্দকোষ, PWA হিসেবে উপলব্ধ |
| **উন্নয়ন কেস স্টাডি** | ১১৭টি দেশ থেকে নথিভুক্ত বাস্তব-বিশ্বের প্রোগ্রাম |
| **DevDiscourse** | অনুশীলনকারীদের প্রাসঙ্গিকতার জন্য নির্বাচিত কিউরেটেড একাডেমিক গবেষণাপত্র বা বই |
| **Dataverse** | ২৪৭টি উন্নয়ন ডেটাসেট, API, টুল এবং MCP সার্ভারের সংগ্রহ |
| **BookSummary** | অধ্যায় নেভিগেশন, প্রমাণ অন্বেষক এবং AI প্রশ্নোত্তর সহ বইয়ের ইন্টারেক্টিভ সঙ্গী |
| **শিক্ষণ ট্র্যাক** | ভূমিকা/আগ্রহ অনুযায়ী সংগঠিত বিষয়বস্তু (যেমন, MEL ও গবেষণা, অর্থনীতি ও নীতি) |
| **শিক্ষণ পথ** | একটি নির্দিষ্ট শিক্ষণ লক্ষ্যের জন্য কোর্স, গেম এবং ল্যাবের কিউরেটেড ক্রম |
| **শিক্ষণ স্ট্রিক** | একজন ব্যবহারকারী প্ল্যাটফর্মের সাথে জড়িত থাকা ধারাবাহিক দিনের সংখ্যা |

## সাবস্ক্রিপশন শব্দাবলী

| শব্দ | সংজ্ঞা |
|------|-----------|
| **Explorer** | বিনামূল্যে স্তর — সকল মূল বিষয়বস্তুতে অ্যাক্সেস |
| **Practitioner** | ₹৩৯৯/মাস — RQ Builder Pro, সার্টিফিকেট, কমিউনিটি অ্যাক্সেস যোগ করে |
| **Professional** | ₹৯৯৯/মাস — সকল ৭টি প্রিমিয়াম টুল, অগ্রাধিকার কোচিং |
| **Organisation** | ₹১,৪৯৯/ব্যবহারকারী/মাস — টিম ড্যাশবোর্ড, কাস্টম পথ, ব্র্যান্ডেড সার্টিফিকেট |
| **কোহর্ট** | তালিকাভুক্তি, সময়সীমা এবং আলোচনা থ্রেড সহ নির্ধারিত গ্রুপ শিক্ষণ সেশন |

## উন্নয়ন ও গবেষণা শব্দাবলী

| শব্দ | সংজ্ঞা |
|------|-----------|
| **MEL** | Monitoring, Evaluation, and Learning (পর্যবেক্ষণ, মূল্যায়ন এবং শিক্ষণ) |
| **ToC** | Theory of Change — কার্যকলাপ কীভাবে উদ্দিষ্ট ফলাফলে নিয়ে যায় তার একটি মডেল |
| **RCT** | Randomised Controlled Trial — প্রভাব মূল্যায়নের জন্য পরীক্ষামূলক ডিজাইন |
| **Logframe** | Logical Framework — কার্যকলাপ, আউটপুট, ফলাফল এবং প্রভাব সংযুক্ত করার একটি পরিকল্পনা সরঞ্জাম |
| **RQ** | Research Question — একটি গবেষণা যে কেন্দ্রীয় প্রশ্নের উত্তর দিতে চায় |
| **DHS** | Demographic and Health Surveys — বহু-দেশ পরিবার সমীক্ষা কর্মসূচি |
| **NFHS** | National Family Health Survey — ভারতের DHS সমতুল্য |
| **NSSO** | National Sample Survey Office — ভারতের জাতীয় পরিসংখ্যান সমীক্ষা সংস্থা |
| **MCP** | Model Context Protocol — AI টুলকে ডেটা উৎসের সাথে সংযুক্ত করার মান |
| **PWA** | Progressive Web App — অফলাইন সমর্থন সহ ডিভাইসে ইনস্টলযোগ্য ওয়েব অ্যাপ্লিকেশন |

## প্রযুক্তিগত শব্দাবলী

| শব্দ | সংজ্ঞা |
|------|-----------|
| **Edge Function** | CDN এজে চলা সার্ভার-সাইড ফাংশন (Netlify/Supabase) |
| **JWT** | JSON Web Token — স্টেটলেস প্রমাণীকরণের জন্য স্বাক্ষরিত টোকেন |
| **RLS** | Row-Level Security — প্রতি ব্যবহারকারী ডেটা অ্যাক্সেস সীমাবদ্ধ করার Supabase/PostgreSQL বৈশিষ্ট্য |
| **Auth Gate** | প্রিমিয়াম টুল সাইটে অ্যাক্সেস টোকেন যাচাই করা Edge Function |
| **Anon Key** | ক্লায়েন্ট-সাইড ব্যবহারের জন্য নিরাপদ Supabase পাবলিক কী (RLS প্রয়োগ করে) |
| **Service Role Key** | RLS বাইপাস করা Supabase অ্যাডমিন কী (কেবল সার্ভার-সাইড) |
