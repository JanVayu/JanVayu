# Dataverse গাইড

ImpactMojo Dataverse হল World Bank, DHS, NFHS এবং NSSO সহ প্রধান উৎস থেকে ২৪৭টি উন্নয়ন ডেটাসেট, API, টুল এবং MCP সার্ভারের একটি কিউরেটেড সংগ্রহ।

---

## Dataverse কী?

Dataverse অনুশীলনকারী, গবেষক এবং ছাত্রদের ডজন ডজন পৃথক পোর্টাল নেভিগেট না করেই উন্নয়ন ডেটা খুঁজে পেতে এবং ব্যবহার করতে সাহায্য করে। প্রতিটি এন্ট্রিতে অন্তর্ভুক্ত:
- **বিবরণ** — ডেটাসেট কী কভার করে
- **উৎস** — মূল প্রতিষ্ঠান
- **অ্যাক্সেস পদ্ধতি** — সরাসরি ডাউনলোড, API, বা MCP সার্ভার
- **ট্যাগ** — দেশ, বিষয়, পদ্ধতি, ফ্রিকোয়েন্সি

---

## ডেটা উৎস

| উৎস | ধরন | কভারেজ |
|--------|------|---------|
| World Bank Open Data | API + ডাউনলোড | বৈশ্বিক, ২০০+ দেশ |
| DHS (Demographic and Health Surveys) | ডাউনলোড | ৯০+ দেশ |
| NFHS (National Family Health Survey) | ডাউনলোড | ভারত, রাজ্য-স্তর |
| NSSO (National Sample Survey Office) | ডাউনলোড | ভারত, জাতীয়/রাজ্য |
| UDISE+ (School Education) | ডাউনলোড | ভারত, জেলা-স্তর |
| ASER (Annual Status of Education Report) | ডাউনলোড | ভারত, গ্রামীণ জেলা |
| RBI (Reserve Bank of India) | API + ডাউনলোড | ভারত, সামষ্টিক অর্থনৈতিক |
| Census of India | ডাউনলোড | ভারত, গ্রাম/ওয়ার্ড-স্তর |

## Dataverse ব্যবহার

### বিষয় অনুযায়ী ব্রাউজ করুন

ডেটাসেটগুলি বিভিন্ন বিষয়ে ট্যাগ করা আছে:
- স্বাস্থ্য ও পুষ্টি
- শিক্ষা
- দারিদ্র্য ও বৈষম্য
- শাসন ও প্রতিষ্ঠান
- পরিবেশ ও জলবায়ু
- জেন্ডার ও সামাজিক অন্তর্ভুক্তি
- অর্থনৈতিক উন্নয়ন
- কৃষি ও গ্রামীণ

### অনুসন্ধান

কীওয়ার্ড, উৎস বা দেশ অনুযায়ী ডেটাসেট খুঁজতে সার্চ বার বা **Ctrl+K** ব্যবহার করুন।

### MCP সার্ভার

নির্বাচিত ডেটাসেটগুলি AI টুল এবং স্বয়ংক্রিয় ওয়ার্কফ্লোর সাথে ইন্টিগ্রেশনের জন্য MCP (Model Context Protocol) সার্ভার হিসেবে উপলব্ধ।

## কর্মশালায়

| কার্যকলাপ | কীভাবে Dataverse ব্যবহার করবেন |
|----------|---------------------|
| ডেটা অন্বেষণ অনুশীলন | দলগুলিকে একটি বিষয় দিন; প্রতিটি দল একটি প্রাসঙ্গিক ডেটাসেট খুঁজে বের করে এবং উপস্থাপন করে |
| গবেষণা ডিজাইন ল্যাব | গবেষণা প্রশ্নের জন্য সেকেন্ডারি ডেটা উৎস চিহ্নিত করতে Dataverse ব্যবহার করুন |
| ভিজ্যুয়ালাইজেশন অনুশীলন | একটি ডেটাসেট ডাউনলোড করুন এবং Viz Cookbook ব্যবহার করে চার্ট তৈরি করুন |
| নীতি বিশ্লেষণ | World Bank বা NFHS ডেটা ব্যবহার করে রাজ্য/দেশ জুড়ে সূচক তুলনা করুন |
