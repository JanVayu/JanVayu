# প্ল্যাটফর্ম পরিদর্শন

ImpactMojo হল দক্ষিণ এশিয়ার জন্য উন্নয়ন অর্থনীতি এবং নীতি শিক্ষা প্রদানকারী একটি বিনামূল্যের শিক্ষামূলক প্ল্যাটফর্ম। সংস্করণ 10.8.5 বর্তমানে [impactmojo.in](https://www.impactmojo.in)-এ লাইভ আছে।

---

## বিষয়বস্তুর ধরন

| ২ | **ভিত্তিমূলক কোর্স** | ৩৯ | স্ব-গতিসম্পন্ন দক্ষতা-কেন্দ্রিক উপকরণ; ২৩টি ভারতীয় লোকশিল্প চিত্রসহ ভিজ্যুয়াল 101 কোর্স ডেক হিসেবে উপলব্ধ |

| ৪ | **ইন্টারেক্টিভ ল্যাব** | ১৯ | হাতে-কলমে ওয়ার্কবেঞ্চ যা বাস্তব আউটপুট তৈরি করে — ToC ডায়াগ্রাম, গবেষণা প্রশ্ন, কোডেড গুণগত ডেটা |

| ৭ | **উন্নয়ন কেস স্টাডি** | ২০০ | ১১৭টি দেশ থেকে বাস্তব-বিশ্বের প্রোগ্রাম, দেশ, বিষয় এবং পদ্ধতি অনুযায়ী ট্যাগ করা |
| ৮ | **DevDiscourses** | ৫০০+ | অনুশীলনকারীদের প্রাসঙ্গিকতার জন্য নির্বাচিত কিউরেটেড একাডেমিক গবেষণাপত্র এবং বই |
| ৯ | **Dataverse** | ২৪৭ | World Bank, DHS, NFHS, NSSO থেকে উন্নয়ন ডেটাসেট, API, টুল এবং MCP সার্ভার |
| ১০ | **বই সারসংক্ষেপ** | ৫ | অধ্যায় নেভিগেশন, প্রমাণ অন্বেষক, ডেটা প্লেগ্রাউন্ড এবং AI প্রশ্নোত্তর সহ ইন্টারেক্টিভ সঙ্গী |

## শিক্ষণ ট্র্যাক

বিষয়বস্তু ভূমিকা এবং আগ্রহ অনুযায়ী ছয়টি ট্র্যাকে সংগঠিত:

| ট্র্যাক | ফোকাস এলাকা |
|-------|-----------|
| MEL ও গবেষণা | পর্যবেক্ষণ, মূল্যায়ন, শিক্ষণ এবং গবেষণা পদ্ধতি |
| অর্থনীতি ও নীতি | উন্নয়ন অর্থনীতি, জননীতি, রাজস্ব বিশ্লেষণ |
| জেন্ডার ও সমতা | জেন্ডার বিশ্লেষণ, আন্তঃবিভাগীয়তা, সমতা কাঠামো |
| শাসন ও সমাজ | শাসনব্যবস্থা, প্রতিষ্ঠান, নাগরিক সম্পৃক্ততা |
| স্বাস্থ্য ও সুস্থতা | জনস্বাস্থ্য, পুষ্টি, সুস্থতা সূচক |
| যোগাযোগ ও ডেটা | ডেটা সাক্ষরতা, ভিজ্যুয়ালাইজেশন, উন্নয়ন যোগাযোগ |

| স্তর | মূল্য | প্রধান বৈশিষ্ট্য |
|------|-------|-------------|
| **Explorer** | চিরকাল বিনামূল্যে | সকল কোর্স, গেম, ল্যাব, হ্যান্ডআউট, ImpactLex, কেস স্টাডি, Dataverse |
| **Practitioner** | ₹৩৯৯/মাস | Research Question Builder Pro, সার্টিফিকেট, কমিউনিটি অ্যাক্সেস |
| **Professional** | ₹৯৯৯/মাস | সকল প্রিমিয়াম টুল (Qual Insights Lab, Statistical Code Converter, VaniScribe, DevData, DevEcon Toolkit), অগ্রাধিকার কোচিং |
| **Organization** | ₹১,৪৯৯/ব্যবহারকারী/মাস | টিম ড্যাশবোর্ড, কাস্টম শিক্ষণ পথ, ব্র্যান্ডেড সার্টিফিকেট, ডেডিকেটেড অ্যাকাউন্ট ম্যানেজার |

## ইন্টারেক্টিভ ডেমো

> **আসন্ন** — সম্পূর্ণ ImpactMojo প্ল্যাটফর্মের একটি ইন্টারেক্টিভ ওয়াকথ্রু এখানে এম্বেড করা হবে। আপনি ডকুমেন্টেশন না ছেড়ে কোর্স ব্রাউজ করতে, গেম চেষ্টা করতে এবং Dataverse অন্বেষণ করতে পারবেন।

## অফলাইন সমর্থন

ImpactMojo একটি Progressive Web App। অফলাইন অ্যাক্সেসের জন্য বিষয়বস্তু ক্যাশে করতে এটি আপনার ডিভাইসে ইনস্টল করুন। অ্যাকাউন্ট বৈশিষ্ট্য এবং গেমগুলির জন্য ইন্টারনেট সংযোগ প্রয়োজন।
