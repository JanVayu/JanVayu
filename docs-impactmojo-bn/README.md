# ImpactMojo ডকুমেন্টেশন

**ImpactMojo** হল দক্ষিণ এশিয়ায় আন্তর্জাতিক উন্নয়ন, সামাজিক প্রভাব এবং জননীতি নিয়ে কাজ করা মানুষদের জন্য একটি বিনামূল্যের শিক্ষা প্ল্যাটফর্ম। এটি কোর্স, গেম, ল্যাব, হ্যান্ডআউট, কেস স্টাডি এবং ডেটা টুল সরবরাহ করে — সব কিছু পেওয়াল বা লগইন প্রয়োজনীয়তা ছাড়াই।

> **লাইভ প্ল্যাটফর্ম:** [www.impactmojo.in](https://www.impactmojo.in)
> **GitHub:** [github.com/JanVayu/ImpactMojo](https://github.com/JanVayu/ImpactMojo)

---

## ImpactMojo কী?

দক্ষিণ এশিয়ায় উন্নয়ন কর্মী, গবেষক, এনজিও কর্মী এবং ছাত্রদের MEL, গবেষণা পদ্ধতি, অর্থনীতি এবং নীতি বিষয়ে সহজলভ্য, উচ্চমানের শিক্ষা প্রয়োজন। ImpactMojo এটি সরবরাহ করে ৪৮টি কোর্স, ১৬টি গেম, ১৯টি ল্যাব, ৪০০+ হ্যান্ডআউট, ২০০টি কেস স্টাডি, ৫০০+ কিউরেটেড গবেষণাপত্র এবং ২৪৭টি ডেটাসেটের মাধ্যমে — সব কিছু বিনামূল্যে।

## মূলনীতি

- **ব্যবহারের সময় বিনামূল্যে** — মূল বিষয়বস্তুতে কোনো লগইন বা পেমেন্ট প্রয়োজন নেই
- **অনুশীলনকারী-প্রথম** — যারা কাজ করছেন তাদের জন্য ডিজাইন করা, শুধু পড়াশোনার জন্য নয়
- **দক্ষিণ এশিয়া-কেন্দ্রিক** — এই অঞ্চলের কেস স্টাডি, ডেটা এবং প্রসঙ্গ
- **সহজলভ্য** — ধীর সংযোগ, পুরনো ডিভাইস এবং PWA-র মাধ্যমে অফলাইনে কাজ করে
- **উন্মুক্ত** — কোড MIT লাইসেন্সের অধীনে; শিক্ষামূলক বিষয়বস্তু CC BY-NC-ND 4.0
