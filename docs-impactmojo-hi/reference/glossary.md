# शब्दावली

ImpactMojo के प्लेटफ़ॉर्म और प्रलेखन में उपयोग किए जाने वाले प्रमुख शब्द।

---

## प्लेटफ़ॉर्म शब्द

| शब्द | परिभाषा |
|------|-----------|
| **फ़्लैगशिप पाठ्यक्रम** | ~13 मॉड्यूल, शब्दावली, केस स्टडी, और चिंतन प्रश्नों के साथ गहन कार्यक्रम |
| **आधारभूत पाठ्यक्रम** | छोटा, स्व-गति कौशल-केंद्रित पाठ्यक्रम (30-60 मिनट) |
| **कोर्स डेक** | भारतीय लोक कला चित्रों के साथ विज़ुअल 101 स्लाइड प्रस्तुति (23 उपलब्ध) |
| **इंटरैक्टिव खेल** | वास्तविक मापदंडों का उपयोग करने वाला ब्राउज़र-आधारित आर्थिक अनुकरण, चर्चा के लिए डिज़ाइन किया गया |
| **इंटरैक्टिव प्रयोगशाला** | ठोस परिणाम उत्पन्न करने वाली व्यावहारिक कार्यशाला (ToC आरेख, शोध प्रश्न, आदि) |
| **हैंडआउट** | एक-पृष्ठ संदर्भ सामग्री — चीट शीट या ब्रीफ़िंग नोट |
| **ImpactLex** | 500+ विकास शब्दों की खोज योग्य शब्दावली, PWA के रूप में उपलब्ध |
| **विकास केस स्टडी** | 117 देशों से प्रलेखित वास्तविक कार्यक्रम |
| **DevDiscourse** | कार्यकर्ता प्रासंगिकता के लिए चयनित क्यूरेटेड शैक्षणिक शोधपत्र या पुस्तक |
| **Dataverse** | 247 विकास डेटासेट, API, उपकरण, और MCP सर्वरों का संग्रह |
| **BookSummary** | अध्याय नेविगेशन, साक्ष्य एक्सप्लोरर, और AI प्रश्नोत्तर के साथ पुस्तक का इंटरैक्टिव साथी |
| **लर्निंग ट्रैक** | भूमिका/रुचि के अनुसार व्यवस्थित सामग्री (जैसे, MEL और शोध, अर्थशास्त्र और नीति) |
| **लर्निंग पाथ** | एक विशिष्ट शिक्षण लक्ष्य के लिए पाठ्यक्रमों, खेलों, और प्रयोगशालाओं का क्यूरेटेड अनुक्रम |
| **लर्निंग स्ट्रीक** | उपयोगकर्ता द्वारा प्लेटफ़ॉर्म पर लगातार दिनों की गणना |

| **कोहोर्ट** | नामांकन, समय सीमा, और चर्चा सूत्रों के साथ अनुसूचित समूह शिक्षण सत्र |

## विकास और शोध शब्द

| शब्द | परिभाषा |
|------|-----------|
| **MEL** | Monitoring, Evaluation, and Learning (निगरानी, मूल्यांकन, और सीखना) |
| **ToC** | Theory of Change — गतिविधियों से इच्छित परिणामों तक का मार्ग मैप करने वाला मॉडल |
| **RCT** | Randomised Controlled Trial — प्रभाव मूल्यांकन के लिए प्रयोगात्मक डिज़ाइन |
| **Logframe** | Logical Framework — गतिविधियों, आउटपुट, परिणामों, और प्रभाव को जोड़ने वाला योजना उपकरण |
| **RQ** | Research Question — वह केंद्रीय प्रश्न जिसका अध्ययन उत्तर देना चाहता है |
| **DHS** | Demographic and Health Surveys — बहु-देशीय घरेलू सर्वेक्षण कार्यक्रम |
| **NFHS** | National Family Health Survey — भारत का DHS समकक्ष |
| **NSSO** | National Sample Survey Office — भारत का राष्ट्रीय सांख्यिकीय सर्वेक्षण निकाय |
| **MCP** | Model Context Protocol — AI उपकरणों को डेटा स्रोतों से जोड़ने का मानक |
| **PWA** | Progressive Web App — ऑफ़लाइन समर्थन के साथ उपकरणों पर इंस्टॉल करने योग्य वेब एप्लिकेशन |

## तकनीकी शब्द

| शब्द | परिभाषा |
|------|-----------|
| **Edge Function** | CDN एज पर चलने वाला सर्वर-साइड फ़ंक्शन (Netlify/Supabase) |
| **JWT** | JSON Web Token — स्टेटलेस प्रमाणीकरण के लिए हस्ताक्षरित टोकन |
| **RLS** | Row-Level Security — Supabase/PostgreSQL सुविधा जो प्रति उपयोगकर्ता डेटा पहुँच को प्रतिबंधित करती है |
| **Auth Gate** | प्रीमियम उपकरण साइटों पर Edge Function जो पहुँच टोकन सत्यापित करता है |
| **Anon Key** | Supabase सार्वजनिक कुंजी जो क्लाइंट-साइड उपयोग के लिए सुरक्षित है (RLS लागू करती है) |
| **Service Role Key** | Supabase व्यवस्थापक कुंजी जो RLS को बायपास करती है (केवल सर्वर-साइड) |
