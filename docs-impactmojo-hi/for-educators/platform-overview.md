| ट्रैक | फ़ोकस क्षेत्र |

| लिंग और समानता | लिंग विश्लेषण, अंतर्विभागीयता, समानता ढाँचे |
| शासन और समाज | शासन, संस्थान, नागरिक भागीदारी |
| स्वास्थ्य और कल्याण | सार्वजनिक स्वास्थ्य, पोषण, कल्याण संकेतक |
| संचार और डेटा | डेटा साक्षरता, विज़ुअलाइज़ेशन, विकास संचार |
