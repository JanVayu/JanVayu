# Dataverse गाइड

ImpactMojo Dataverse World Bank, DHS, NFHS, और NSSO सहित प्रमुख स्रोतों से 247 विकास डेटासेट, API, उपकरण, और MCP सर्वरों का एक क्यूरेटेड संग्रह है।

---

## Dataverse क्या है?

Dataverse कार्यकर्ताओं, शोधकर्ताओं, और छात्रों को दर्जनों अलग-अलग पोर्टलों को नेविगेट किए बिना विकास डेटा खोजने और उपयोग करने में मदद करता है। प्रत्येक प्रविष्टि में शामिल है:
- **विवरण** — डेटासेट क्या कवर करता है
- **स्रोत** — मूल संस्थान
- **पहुँच विधि** — प्रत्यक्ष डाउनलोड, API, या MCP सर्वर
- **टैग** — देश, विषय, कार्यप्रणाली, आवृत्ति

## डेटा स्रोत

| स्रोत | प्रकार | कवरेज |
|--------|------|---------|
| World Bank Open Data | API + डाउनलोड | वैश्विक, 200+ देश |
| DHS (Demographic and Health Surveys) | डाउनलोड | 90+ देश |
| NFHS (National Family Health Survey) | डाउनलोड | भारत, राज्य-स्तरीय |
| NSSO (National Sample Survey Office) | डाउनलोड | भारत, राष्ट्रीय/राज्य |
| UDISE+ (स्कूल शिक्षा) | डाउनलोड | भारत, जिला-स्तरीय |
| ASER (Annual Status of Education Report) | डाउनलोड | भारत, ग्रामीण जिले |
| RBI (भारतीय रिज़र्व बैंक) | API + डाउनलोड | भारत, समष्टि आर्थिक |
| भारत की जनगणना | डाउनलोड | भारत, गाँव/वार्ड-स्तरीय |

## Dataverse का उपयोग

### विषय के अनुसार ब्राउज़ करें

डेटासेट निम्नलिखित विषयों में टैग किए गए हैं:
- स्वास्थ्य और पोषण
- शिक्षा
- गरीबी और असमानता
- शासन और संस्थान
- पर्यावरण और जलवायु
- लिंग और सामाजिक समावेशन
- आर्थिक विकास
- कृषि और ग्रामीण

### खोज

कीवर्ड, स्रोत, या देश के अनुसार डेटासेट खोजने के लिए खोज बार या **Ctrl+K** का उपयोग करें।

### MCP सर्वर

चयनित डेटासेट AI उपकरणों और स्वचालित कार्यप्रवाहों के साथ एकीकरण के लिए MCP (Model Context Protocol) सर्वर के रूप में उपलब्ध हैं।

## कार्यशालाओं में

| गतिविधि | Dataverse का उपयोग कैसे करें |
|----------|---------------------|
| डेटा अन्वेषण अभ्यास | टीमों को एक विषय दें; प्रत्येक टीम एक प्रासंगिक डेटासेट खोजे और प्रस्तुत करे |
| शोध डिज़ाइन प्रयोगशाला | शोध प्रश्नों के लिए द्वितीयक डेटा स्रोतों की पहचान करने हेतु Dataverse का उपयोग करें |
| विज़ुअलाइज़ेशन अभ्यास | एक डेटासेट डाउनलोड करें और Viz Cookbook का उपयोग करके चार्ट बनाएँ |
| नीति विश्लेषण | World Bank या NFHS डेटा का उपयोग करके राज्यों/देशों के बीच संकेतकों की तुलना करें |
