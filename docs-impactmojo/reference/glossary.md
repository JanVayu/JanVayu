# Glossary

Key terms used across ImpactMojo's platform and documentation.

---

## Platform Terms

| Term | Definition |
|------|-----------|
| **Flagship Course** | Deep-dive programme with ~13 modules, lexicon, case studies, and reflection prompts |
| **Foundational Course** | Shorter, self-paced skill-focused course (30–60 minutes) |
| **Course Deck** | Visual 101 slide presentation with Indian folk art illustrations (23 available) |
| **Interactive Game** | Browser-based economic simulation using real parameters, designed for debrief |
| **Interactive Lab** | Hands-on workbench producing a tangible output (ToC diagram, research question, etc.) |
| **Handout** | Single-page reference material — cheat sheet or briefing note |
| **ImpactLex** | Searchable glossary of 500+ development terms, available as PWA |
| **Dev Case Study** | Real-world programme documented from 117 countries |
| **DevDiscourse** | Curated academic paper or book selected for practitioner relevance |
| **Dataverse** | Collection of 247 development datasets, APIs, tools, and MCP servers |
| **BookSummary** | Interactive companion for a book with chapter navigation, evidence explorer, and AI Q&A |
| **Learning Track** | Content organised by role/interest (e.g., MEL & Research, Economics & Policy) |
| **Learning Path** | Curated sequence of courses, games, and labs for a specific learning goal |
| **Learning Streak** | Count of consecutive days a user engages with the platform |

## Development & Research Terms

| Term | Definition |
|------|-----------|
| **MEL** | Monitoring, Evaluation, and Learning |
| **ToC** | Theory of Change — a model mapping how activities lead to intended outcomes |
| **RCT** | Randomised Controlled Trial — experimental design for impact evaluation |
| **Logframe** | Logical Framework — a planning tool linking activities, outputs, outcomes, and impact |
| **RQ** | Research Question — the central question a study aims to answer |
| **DHS** | Demographic and Health Surveys — multi-country household survey programme |
| **NFHS** | National Family Health Survey — India's DHS equivalent |
| **NSSO** | National Sample Survey Office — India's national statistical survey body |
| **MCP** | Model Context Protocol — standard for connecting AI tools to data sources |
| **PWA** | Progressive Web App — web application installable on devices with offline support |

## Technical Terms

| Term | Definition |
|------|-----------|
| **Edge Function** | Server-side function running at CDN edge (Netlify/Supabase) |
| **JWT** | JSON Web Token — signed token for stateless authentication |
| **RLS** | Row-Level Security — Supabase/PostgreSQL feature restricting data access per user |
| **Auth Gate** | Edge Function on premium tool sites that validates access tokens |
| **Anon Key** | Supabase public key safe for client-side use (enforces RLS) |
| **Service Role Key** | Supabase admin key that bypasses RLS (server-side only) |
