# Handouts Guide

ImpactMojo provides 400+ single-page reference materials — well-designed cheat sheets and briefing notes that are printable and shareable.

---

## What Are Handouts?

Handouts are concise, single-page documents covering a specific concept, framework, or method. Each handout is designed to:
- Stand alone as a quick reference
- Complement course modules and lab work
- Be printed for offline distribution in workshops
- Be shared digitally via link or download

## Using Handouts in Workshops

| Use Case | When | How |
|----------|------|-----|
| Pre-reading | Before the session | Assign 2–3 handouts to level-set participants |
| In-session reference | During activities | Print and distribute as quick-reference cards |
| Lab companion | During labs | Pair with lab exercises as methodology reminders |
| Take-home | After the session | Share download links for continued reference |

## Creating Custom Handouts

Contributors can submit new handouts following these guidelines:
- One concept per page
- Plain language, minimal jargon
- Include a visual (diagram, flowchart, or table) where possible
- Cite primary sources
- Available in English; translations welcome

See [How to Contribute](../contributing/how-to-contribute.md) for submission details.
