# Dataverse Guide

The ImpactMojo Dataverse is a curated collection of 247 development datasets, APIs, tools, and MCP servers from major sources including the World Bank, DHS, NFHS, and NSSO.

---

## What is the Dataverse?

The Dataverse helps practitioners, researchers, and students find and use development data without navigating dozens of separate portals. Each entry includes:
- **Description** — what the dataset covers
- **Source** — the originating institution
- **Access method** — direct download, API, or MCP server
- **Tags** — country, theme, methodology, frequency

---

## Data Sources

| Source | Type | Coverage |
|--------|------|---------|
| World Bank Open Data | API + Download | Global, 200+ countries |
| DHS (Demographic and Health Surveys) | Download | 90+ countries |
| NFHS (National Family Health Survey) | Download | India, state-level |
| NSSO (National Sample Survey Office) | Download | India, national/state |
| UDISE+ (School Education) | Download | India, district-level |
| ASER (Annual Status of Education Report) | Download | India, rural districts |
| RBI (Reserve Bank of India) | API + Download | India, macroeconomic |
| Census of India | Download | India, village/ward-level |

## Using the Dataverse

### Browse by Theme

Datasets are tagged across themes including:
- Health & Nutrition
- Education
- Poverty & Inequality
- Governance & Institutions
- Environment & Climate
- Gender & Social Inclusion
- Economic Development
- Agriculture & Rural

### Search

Use the search bar or **Ctrl+K** to find datasets by keyword, source, or country.

### MCP Servers

Select datasets are available as MCP (Model Context Protocol) servers for integration with AI tools and automated workflows.
