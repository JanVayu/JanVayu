# Platform Overview

ImpactMojo is a free educational platform providing development economics and policy education for South Asia. Version 10.8.5 is currently live at [impactmojo.in](https://www.impactmojo.in).

---

## Content Types

| # | Content Type | Count | Description |
|---|-------------|-------|-------------|
| 1 | **Flagship Courses** | 9 | Deep-dive programs with ~13 modules each, lexicons of 50–65 key terms, South Asian case studies, reflection prompts, and curated readings |
| 2 | **Foundational Courses** | 39 | Self-paced skill-focused materials; 23 available as visual 101 Course Decks with Indian folk art illustrations |
| 3 | **Interactive Games** | 16 | Browser-based economic simulations using real economic parameters, designed for post-game debriefing |
| 4 | **Interactive Labs** | 19 | Hands-on workbenches producing tangible outputs — ToC diagrams, research questions, coded qualitative data |
| 5 | **Handouts** | 400+ | Single-page reference materials — cheat sheets and briefing notes, printable and shareable |
| 6 | **ImpactLex** | 500+ terms | Searchable glossary available as a Progressive Web App for offline access |
| 7 | **Dev Case Studies** | 200 | Real-world programs from 117 countries, tagged by country, theme, and methodology |
| 8 | **DevDiscourses** | 500+ | Curated academic papers and books selected for practitioner relevance |
| 9 | **Dataverse** | 247 | Development datasets, APIs, tools, and MCP servers from World Bank, DHS, NFHS, NSSO |
| 10 | **Book Summaries** | 5 | Interactive companions with chapter navigation, evidence explorers, data playgrounds, and AI Q&A |

## Learning Tracks

Content is organised into six tracks by role and interest:

| Track | Focus Area |
|-------|-----------|
| MEL & Research | Monitoring, evaluation, learning, and research methods |
| Economics & Policy | Development economics, public policy, fiscal analysis |
| Gender & Equity | Gender analysis, intersectionality, equity frameworks |
| Governance & Society | Governance, institutions, civic engagement |
| Health & Wellbeing | Public health, nutrition, wellbeing indicators |
| Communication & Data | Data literacy, visualisation, development communication |

## Personal Learning Tools

| Tool | Description |
|------|-------------|
| **Bookmarks** | Save favourite courses and resources |
| **Compare** | Side-by-side course comparison |
| **Personal Notes** | Capture reflections while reading modules |
| **Learning Streak** | Consecutive day tracking with milestone badges |
| **Pomodoro Timer** | Built-in focus session tracking |
| **Analytics** | Personal learning statistics dashboard |
| **Lo-Fi Playlists** | Spotify and YouTube study music integration |

## Pricing Tiers

| Tier | Price | Key Features |
|------|-------|-------------|
| **Explorer** | Free forever | All courses, games, labs, handouts, ImpactLex, case studies, Dataverse |
| **Practitioner** | ₹399/month | Research Question Builder Pro, certificates, community access |
| **Professional** | ₹999/month | All premium tools (Qual Insights Lab, Statistical Code Converter, VaniScribe, DevData, DevEcon Toolkit), priority coaching |
| **Organization** | ₹1,499/user/month | Team dashboards, custom learning pathways, branded certificates, dedicated account manager |

Annual subscriptions include a 2-month discount.

## Offline Support

ImpactMojo is a Progressive Web App. Install it on your device to cache content for offline access. Account features and games require an internet connection.
