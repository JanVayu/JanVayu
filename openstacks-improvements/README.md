# OpenStacks for Change — Ecosystem Improvement Kit

This directory contains improved READMEs, governance docs, and descriptions for all 9 repositories in the [OpenStacks for Change](https://openstacks.dev) ecosystem.

## How to Use

Copy each repo's `README.md` into the corresponding GitHub repository, replacing the existing one. The `shared/` folder contains governance documents that should be identical across all repos.

## Repository Map

```
openstacks.dev (Hub)
│
├── CONTENT STACKS (real tools & code)
│   ├── InsightStack ── MEL tools, calculators, research docs (Stata/Python/R)
│   ├── FieldStack ──── Applied R notebooks for fieldwork & evaluation
│   └── EquityStack ─── Python/Jupyter workflows for development data
│
├── KNOWLEDGE STACK
│   └── SignalStack ─── Research Rundown newsletter companion
│
├── INFRASTRUCTURE STACKS (planned/early-stage)
│   ├── RootStack ───── Database schemas & seed data
│   ├── BridgeStack ─── API backend (FastAPI)
│   └── ViewStack ───── Frontend UI
│
└── DOMAIN STACK (planned)
    └── PolicyStack ─── South Asia policy tracker
```
