# शब्दकोश

ImpactMojo च्या व्यासपीठ आणि दस्तऐवजांमध्ये वापरल्या जाणाऱ्या प्रमुख संज्ञा.

---

## व्यासपीठ संज्ञा

| संज्ञा | व्याख्या |
|------|-----------|
| **प्रमुख अभ्यासक्रम** | ~१३ मॉड्यूल, शब्दकोश, केस स्टडीज आणि चिंतन प्रश्नांसह सखोल कार्यक्रम |
| **पायाभूत अभ्यासक्रम** | लहान, स्वयंगतीने शिकता येणारा कौशल्य-केंद्रित अभ्यासक्रम (३०-६० मिनिटे) |
| **Course Deck** | भारतीय लोककला चित्रांसह दृश्य 101 स्लाइड सादरीकरण (२३ उपलब्ध) |
| **परस्परसंवादी खेळ** | वास्तविक मापदंड वापरणारे ब्राउझर-आधारित आर्थिक सिम्युलेशन, चर्चेसाठी तयार केलेले |
| **परस्परसंवादी प्रयोगशाळा** | ठोस उत्पादन देणारा प्रत्यक्ष कार्यबेंच (ToC आकृती, संशोधन प्रश्न इ.) |
| **हँडआउट** | एकल-पृष्ठ संदर्भ साहित्य — चीट शीट किंवा माहिती नोट |
| **ImpactLex** | ५००+ विकास संज्ञांचा शोधनीय शब्दकोश, PWA म्हणून उपलब्ध |
| **विकास केस स्टडी** | ११७ देशांमधून नोंदवलेला वास्तविक कार्यक्रम |
| **DevDiscourse** | व्यावसायिक प्रासंगिकतेसाठी निवडलेला क्युरेटेड शैक्षणिक शोधनिबंध किंवा पुस्तक |
| **Dataverse** | २४७ विकास डेटासेट्स, API, साधने आणि MCP सर्व्हर्सचे संग्रह |
| **BookSummary** | प्रकरण नेव्हिगेशन, पुरावा एक्सप्लोरर आणि AI प्रश्नोत्तरे असलेला परस्परसंवादी पुस्तक साथीदार |
| **शिक्षण मार्ग** | भूमिका/स्वारस्यानुसार व्यवस्थित आशय (उदा., MEL आणि संशोधन, अर्थशास्त्र आणि धोरण) |
| **शिक्षण पथ** | विशिष्ट शिक्षण ध्येयासाठी क्युरेटेड अभ्यासक्रम, खेळ आणि प्रयोगशाळांचा क्रम |
| **शिक्षण स्ट्रीक** | वापरकर्त्याने व्यासपीठाशी सलग दिवस संलग्न असलेली संख्या |

| संज्ञा | व्याख्या |

| **Practitioner** | ₹३९९/महिना — RQ Builder Pro, प्रमाणपत्रे, समुदाय प्रवेश जोडते |
| **Professional** | ₹९९९/महिना — सर्व ७ प्रीमियम साधने, प्राधान्य मार्गदर्शन |
| **Organisation** | ₹१,४९९/वापरकर्ता/महिना — टीम डॅशबोर्ड्स, सानुकूल मार्ग, ब्रँडेड प्रमाणपत्रे |
| **समूह** | नोंदणी, अंतिम मुदत आणि चर्चा थ्रेड्ससह अनुसूचित गट शिक्षण सत्र |

## विकास आणि संशोधन संज्ञा

| संज्ञा | व्याख्या |
|------|-----------|
| **MEL** | Monitoring, Evaluation, and Learning (देखरेख, मूल्यमापन आणि शिक्षण) |
| **ToC** | Theory of Change — उपक्रम अपेक्षित परिणामांकडे कसे नेतात याचा नकाशा |
| **RCT** | Randomised Controlled Trial — प्रभाव मूल्यमापनासाठी प्रायोगिक रचना |
| **Logframe** | Logical Framework — उपक्रम, उत्पादने, परिणाम आणि प्रभाव जोडणारे नियोजन साधन |
| **RQ** | Research Question — अभ्यासाचा मध्यवर्ती प्रश्न |
| **DHS** | Demographic and Health Surveys — बहु-देशीय घरगुती सर्वेक्षण कार्यक्रम |
| **NFHS** | National Family Health Survey — भारताचा DHS समकक्ष |
| **NSSO** | National Sample Survey Office — भारताची राष्ट्रीय सांख्यिकी सर्वेक्षण संस्था |
| **MCP** | Model Context Protocol — AI साधनांना डेटा स्रोतांशी जोडण्याचे मानक |
| **PWA** | Progressive Web App — ऑफलाइन समर्थनासह उपकरणांवर स्थापित करता येणारा वेब ॲप्लिकेशन |

## तांत्रिक संज्ञा

| संज्ञा | व्याख्या |
|------|-----------|
| **Edge Function** | CDN एज वर चालणारे सर्व्हर-साइड फंक्शन (Netlify/Supabase) |
| **JWT** | JSON Web Token — स्टेटलेस प्रमाणीकरणासाठी स्वाक्षरित टोकन |
| **RLS** | Row-Level Security — प्रत्येक वापरकर्त्याचा डेटा प्रवेश प्रतिबंधित करणारे Supabase/PostgreSQL वैशिष्ट्य |
| **Auth Gate** | प्रीमियम साधन साइट्सवर प्रवेश टोकन सत्यापित करणारे Edge Function |
| **Anon Key** | क्लायंट-साइड वापरासाठी सुरक्षित Supabase सार्वजनिक कळ (RLS लागू करते) |
| **Service Role Key** | RLS बायपास करणारी Supabase प्रशासक कळ (केवळ सर्व्हर-साइड) |
