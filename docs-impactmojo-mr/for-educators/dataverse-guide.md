# Dataverse मार्गदर्शिका

ImpactMojo Dataverse हे World Bank, DHS, NFHS आणि NSSO सारख्या प्रमुख स्रोतांकडून २४७ विकास डेटासेट्स, API, साधने आणि MCP सर्व्हर्सचे क्युरेटेड संग्रह आहे.

---

## Dataverse म्हणजे काय?

Dataverse व्यावसायिक, संशोधक आणि विद्यार्थ्यांना डझनभर स्वतंत्र पोर्टल्स न शोधता विकास डेटा शोधण्यात आणि वापरण्यात मदत करते. प्रत्येक नोंदीमध्ये समाविष्ट आहे:
- **वर्णन** — डेटासेट काय समाविष्ट करतो
- **स्रोत** — मूळ संस्था
- **प्रवेश पद्धत** — थेट डाउनलोड, API किंवा MCP सर्व्हर
- **टॅग** — देश, विषय, पद्धत, वारंवारता

---

## डेटा स्रोत

| स्रोत | प्रकार | व्याप्ती |
|--------|------|---------|
| World Bank Open Data | API + डाउनलोड | जागतिक, २००+ देश |
| DHS (Demographic and Health Surveys) | डाउनलोड | ९०+ देश |
| NFHS (National Family Health Survey) | डाउनलोड | भारत, राज्य-स्तरीय |
| NSSO (National Sample Survey Office) | डाउनलोड | भारत, राष्ट्रीय/राज्य |
| UDISE+ (School Education) | डाउनलोड | भारत, जिल्हा-स्तरीय |
| ASER (Annual Status of Education Report) | डाउनलोड | भारत, ग्रामीण जिल्हे |
| RBI (Reserve Bank of India) | API + डाउनलोड | भारत, समष्टि आर्थिक |
| Census of India | डाउनलोड | भारत, गाव/वॉर्ड-स्तरीय |

## Dataverse चा वापर

### विषयानुसार ब्राउझ करा

डेटासेट्स खालील विषयांमध्ये टॅग केलेले आहेत:
- आरोग्य आणि पोषण
- शिक्षण
- गरिबी आणि असमानता
- शासन आणि संस्था
- पर्यावरण आणि हवामान
- लिंग आणि सामाजिक समावेशन
- आर्थिक विकास
- कृषी आणि ग्रामीण

### शोध

कीवर्ड, स्रोत किंवा देशानुसार डेटासेट्स शोधण्यासाठी शोध बार किंवा **Ctrl+K** वापरा.

### MCP सर्व्हर्स

निवडक डेटासेट्स AI साधने आणि स्वयंचलित कार्यप्रवाहांशी एकत्रीकरणासाठी MCP (Model Context Protocol) सर्व्हर म्हणून उपलब्ध आहेत.

## कार्यशाळांमध्ये

| उपक्रम | Dataverse कसे वापरावे |
|----------|---------------------|
| डेटा अन्वेषण व्यायाम | संघांना एक विषय नियुक्त करा; प्रत्येक संघ संबंधित डेटासेट शोधतो आणि सादर करतो |
| संशोधन रचना प्रयोगशाळा | संशोधन प्रश्नांसाठी दुय्यम डेटा स्रोत ओळखण्यासाठी Dataverse वापरा |
| दृश्यीकरण सराव | डेटासेट डाउनलोड करा आणि Viz Cookbook वापरून चार्ट तयार करा |
| धोरण विश्लेषण | World Bank किंवा NFHS डेटा वापरून राज्य/देशांमधील निर्देशांकांची तुलना करा |
