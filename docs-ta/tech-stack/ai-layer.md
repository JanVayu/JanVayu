# AI அடுக்கு — Google Gemini 2.5 Flash

JanVayu-ன் AI அம்சங்கள் **Google Gemini 2.5 Flash** model-ஆல் இயக்கப்படுகின்றன, Netlify Functions-லிருந்து `@google/generative-ai` npm package வழியாக அணுகப்படுகிறது.

---

## ஏன் Gemini 2.5 Flash?

| அளவுகோல் | தேர்வு |
|-----------|--------|
| **செலவு** | Free tier (Google AI Studio) — பில்லிங் தேவையில்லை |
| **வேகம்** | குறைந்த-தாமத பதில்களுக்கு optimize செய்யப்பட்ட Flash variant |
| **வரம்புகள்** | நாளுக்கு 250 கோரிக்கைகள், நிமிடத்திற்கு 10 — பொது நலன் தளத்திற்கு போதுமானது |
| **பன்மொழி** | வலுவான ஹிந்தி ஆதரவு (JanVayu-ன் பயனர்களுக்கு முக்கியம்) |
| **தரம்** | தரவு அடிப்படையிலான உண்மை பதில்களுக்கு போதுமானது (படைப்பு எழுத்து அல்ல) |

### ஏற்றுக்கொள்ளப்பட்ட சமரசங்கள்

- **GPT-4 / Claude அல்ல** — Gemini-ன் free tier தான் வேறுபடுத்துகிறது. JanVayu சூன்ய பட்ஜெட்டில் இயங்குகிறது.
- **Output token வரம்பு** — அனைத்து prompts-ம் output-ஐ 150-400 tokens-க்கு கட்டுப்படுத்துகின்றன. இது ஒரு அம்சம்: சுருக்கமான பதில்கள் civic தரவுக்கு மிகவும் பயனுள்ளவை.
- **Fine-tuning இல்லை** — Prompt engineering மட்டுமே. ஒவ்வொரு skill-ம் ஒரு system prompt, fine-tuned model அல்ல.

## Integration கட்டமைப்பு

```
Browser (client)
    │
    │ POST /.netlify/functions/air-query
    │ POST /.netlify/functions/health-advisory
    │ POST /.netlify/functions/accountability-brief
    │ GET  /.netlify/functions/anomaly-check
    │
    ▼
Netlify Function (server-side)
    │
    │ 1. பயனர் உள்ளீட்டை parse செய்
    │ 2. WAQI-லிருந்து நேரடி AQI தரவை பெறு
    │ 3. நிஜ தரவுடன் prompt-ஐ உருவாக்கு
    │ 4. Gemini API-ஐ அழை
    │ 5. பதிலை திருப்பு (அல்லது raw தரவுக்கு fallback)
    │
    ▼
Google Gemini API
    │
    │ gemini-2.5-flash model
    │ maxOutputTokens: 150-400
    │ temperature: 0.3-0.7
    │
    ▼
கட்டமைக்கப்பட்ட பதில் → Browser
```

**முக்கிய பாதுகாப்பு கொள்கை:** `GEMINI_API_KEY` ஒருபோதும் கிளையண்டை தொடாது. அனைத்து AI அழைப்புகளும் Netlify Functions வழியாக சர்வர்-பக்கம்.

## நான்கு AI அம்சங்கள்

### 1. JanVayu-விடம் கேளுங்கள் (`air-query.mjs`)
- **உள்ளீடு:** நகரப் பெயர் + தன்னிச்சையான கேள்வி
- **செலுத்தப்படும் சூழல்:** WAQI-லிருந்து நேரடி PM2.5 மற்றும் AQI
- **வெளியீடு:** < 150 சொற்கள், உண்மையான அளவீட்டின் அடிப்படையில்
- **மொழிகள்:** ஆங்கிலம் மற்றும் ஹிந்தி
- **Fallback:** Rate-limited-ஆக இருந்தால் raw AQI தரவை திருப்புகிறது

### 2. சுகாதார ஆலோசனை (`health-advisory.mjs`)
- **உள்ளீடு:** நகரம் + வயது + சுகாதார நிலைமைகள் + வெளிப்புற மணி நேரங்கள்
- **செலுத்தப்படும் சூழல்:** அந்த நகரத்திற்கான நேரடி AQI
- **வெளியீடு:** நிறக்குறியீடு ஆபத்து அளவு + உறுதியான பரிந்துரைகள்
- **Fallback:** பொதுவான WHO-வழிகாட்டுதல் ஆலோசனை

### 3. பொறுப்புணர்வு சுருக்கம் (`accountability-brief.mjs`)
- **உள்ளீடு:** நகரப் பெயர்
- **செலுத்தப்படும் சூழல்:** நேரடி AQI + பருவகால அடிப்படை மதிப்புகள் + GRAP நிலைகள்
- **வெளியீடு:** கட்டமைக்கப்பட்ட சுருக்கம் (தற்போதைய நிலை, NCAP இலக்குகள், 5 பொறுப்புணர்வு கேள்விகள்)
- **Max tokens:** 400
- **Fallback:** Raw data அட்டவணையை திருப்புகிறது

### 4. முரண்பாடு கண்டறிதல் (`anomaly-check.mjs`)
- **உள்ளீடு:** எதுவும் இல்லை (5 பெருநகரங்களை தானாக கண்காணிக்கிறது)
- **வரம்பு:** 2× பருவகால அடிப்படை = முரண்பாடு
- **வெளியீடு:** ஒவ்வொரு முரண்பாட்டுக்கும் ஒரு வாக்கிய AI விளக்கம்
- **Cache:** 10 நிமிட Netlify Blobs cache
- **Fallback:** AI விளக்கம் இல்லாமல் முரண்பாடு flag-ஐ திருப்புகிறது

## Rate Limiting உத்தி

Free-tier Gemini வரம்புகள்:

| வரம்பு | மதிப்பு |
|-------|-------|
| நாளுக்கு கோரிக்கைகள் | 250 |
| நிமிடத்திற்கு கோரிக்கைகள் | 10 |
| நிமிடத்திற்கு Tokens | 1,000,000 |

JanVayu இந்த பட்ஜெட்டை அம்சங்களுக்கு விநியோகிக்கிறது:
- Anomaly check: ~144/நாள் (cached, page load-ல் இயங்குகிறது)
- Air query: ~50/நாள் (பயனர் தொடங்குவது)
- Health advisory: ~30/நாள் (பயனர் தொடங்குவது)
- Accountability brief: ~20/நாள் (பயனர் தொடங்குவது)

வரம்புகள் அடைந்தால், ஒவ்வொரு function-ம் raw data-க்கு அழகாக fallback ஆகிறது. பயனர் எப்போதும் பதிலைப் பெறுகிறார் — AI பகுப்பாய்வு இல்லாமல் மட்டும்.

## Prompt Engineering

அனைத்து Gemini prompts-ம் இந்த கொள்கைகளைப் பின்பற்றுகின்றன:

1. **எப்போதும் நிஜ தரவை செலுத்து** — model நினைவகத்திலிருந்து மட்டும் உருவாக்காது
2. **துல்லியமான output format-ஐ குறிப்பிடு** — குறிப்பாக கட்டமைக்கப்பட்ட சுருக்கங்களுக்கு
3. **சொல் எண்ணிக்கையை கட்டுப்படுத்து** — ஒவ்வொரு prompt-லும் வெளிப்படையான வரம்புகள்
4. **தோல்வி முறையை பெயரிடு** — "பொதுவான ஆலோசனை கொடுக்காதே", "தயங்காதே"
5. **பன்மொழி அறிவுறுத்தல்** — "கேள்வியின் மொழியில் பதிலளி"
6. **அழகான degradation** — ஒவ்வொரு function-க்கும் AI அல்லாத fallback path உள்ளது

முழு prompt ஆவணங்களுக்கு [Skills & AI Prompts](../skills/README.md) பிரிவைப் பார்க்கவும்.
