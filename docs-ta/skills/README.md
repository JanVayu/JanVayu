# Skills & AI Prompts

இந்தப் பிரிவு AI skill files-ஐ ஆவணப்படுத்துகிறது — system prompts, அறிவுறுத்தல் patterns மற்றும் prompt engineering முடிவுகள் — JanVayu-ஐ உருவாக்கவும் இயக்கவும் பயன்படுத்தப்பட்டவை.

இவை சுருக்கமான வழிகாட்டுதல்கள் அல்ல. இவை codebase-ல் உட்பொதிக்கப்பட்ட உண்மையான அறிவுறுத்தல்கள், மேலும் மேம்பாட்டின் போது பயன்படுத்தப்பட்ட பரந்த prompting அணுகுமுறை. JanVayu-ஐ மற்றொரு நகரம் அல்லது சுற்றுச்சூழல் domain-க்கு fork செய்கிறீர்கள் என்றால், இந்தப் பிரிவே தொடங்குமிடம்.

---

## Skill File என்றால் என்ன?

Skill file என்பது ஒரு குறிப்பிட்ட சூழலில் AI model எவ்வாறு நடந்துகொள்ள வேண்டும் என்பதை சொல்லும் கட்டமைக்கப்பட்ட system prompt. "காற்று தரம் பற்றிய கேள்விகளுக்கு பதிலளி" என்ற பொதுவான prompt-க்கு பதிலாக, skill file வரையறுக்கிறது:

- Model-ன் **பங்கு மற்றும் persona**
- சூழலாக பெறும் **தரவு**
- பின்பற்ற வேண்டிய **output format**
- தொனி, நீளம் மற்றும் கூற்றுகள் மீதான **கட்டுப்பாடுகள்**
- தவிர்க்க வேண்டிய **தோல்வி முறைகள்**

JanVayu நான்கு Gemini-இயக்கப்படும் அம்சங்களுக்கும் skill files-ஐ பயன்படுத்துகிறது.

## JanVayu-ல் பயன்படுத்தப்பட்ட பொதுவான Prompting கொள்கைகள்

இந்தக் கொள்கைகள் அனைத்து skill files-ம் எவ்வாறு எழுதப்பட்டன என்பதை வழிநடத்தியது:

1. **நிஜ தரவில் அடிப்படையிடு** — ஒவ்வொரு prompt-ம் உண்மையான நேரடி எண்களைப் பெறுகிறது. Model நினைவகத்திலிருந்து மட்டும் உருவாக்காது.
2. **output format-ஐ துல்லியமாக குறிப்பிடு** — குறிப்பாக பொறுப்புணர்வு சுருக்கத்திற்கு, model உருவாக்க வேண்டிய துல்லியமான அமைப்பு சொல்லப்படுகிறது.
3. **நீளத்தை கட்டுப்படுத்து** — அனைத்து prompts-லும் வெளிப்படையான சொல் அல்லது token வரம்புகள் அதிக உருவாக்கத்தை தடுக்கின்றன.
4. **தோல்வி முறையை பெயரிடு** — prompts model-க்கு என்ன செய்ய கூடாது என்று சொல்கின்றன (எ.கா., "பொதுவான ஆலோசனை கொடுக்காதே", "தயங்காதே").
5. **பன்மொழி இயல்புநிலை** — பொருத்தமான இடங்களில், model பயனரின் மொழியில் பதிலளிக்க அறிவுறுத்தப்படுகிறது.
6. **அழகாக degrade செய்** — ஒவ்வொரு function-க்கும் API அழைப்பு தோல்வியடைந்தால் raw data-ஐ திருப்பும் AI அல்லாத fallback path உள்ளது.
