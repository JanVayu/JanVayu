# தள கண்ணோட்டம்

JanVayu என்பது இந்தியாவின் காற்று தர நெருக்கடியின் ஒவ்வொரு பரிமாணத்தையும் உள்ளடக்கிய 23 பிரிவுகளைக் கொண்ட ஒற்றைப்பக்க வலை பயன்பாடு — நேரடி தரவிலிருந்து சட்ட பொறுப்புணர்வு வரை.
