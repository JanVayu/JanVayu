# குடிமக்கள் செயல் கருவிகள்

JanVayu குடிமக்கள், பத்திரிகையாளர்கள், ஆராய்ச்சியாளர்கள் மற்றும் வாதிடுபவர்களுக்கு காற்று தரத்தில் நடவடிக்கை எடுக்க நடைமுறை கருவிகளை வழங்குகிறது.

---

## RTI வார்ப்புருக்கள்

Right to Information Act (2005)-க்கான முன்-வரையப்பட்ட வார்ப்புருக்கள்:

| வார்ப்புரு | இலக்கு அதிகாரம் |
|----------|-----------------|
| NCAP நிதி பயன்பாடு | மாநில மாசுபாட்டு கட்டுப்பாட்டு வாரியம் |
| GRAP இணக்க அறிக்கை | Commission for Air Quality Management (CAQM) |
| மூல பகுப்பாய்வு ஆய்வு | Central Pollution Control Board (CPCB) |
| தொழிற்சாலை உமிழ்வு தரவு | மாநில PCB / MoEFCC |
| நகர செயல் திட்ட நிலை | மாநகராட்சி |

வார்ப்புருக்கள் **ஆங்கிலம் மற்றும் ஹிந்தியில்** கிடைக்கின்றன, [rtionline.gov.in](https://rtionline.gov.in) RTI ஆன்லைன் போர்ட்டலுக்கு வடிவமைக்கப்பட்டுள்ளன.

---

## வாதக்கலை வழிகாட்டிகள்

படிப்படியான வழிகாட்டிகள்:

- **RTI எவ்வாறு தாக்கல் செய்வது** — சமர்ப்பிப்பு முதல் முதல் மேல்முறையீடு வரை
- **உங்கள் வார்டு கவுன்சிலரை எவ்வாறு அணுகுவது** — வார்டு அளவிலான பொறுப்புணர்வு சுருக்க உருவாக்கியைப் பயன்படுத்தி
- **ஊடகங்களை எவ்வாறு ஈடுபடுத்துவது** — தரவுகளுடன் காற்று தர கதையை வடிவமைத்தல்
- **பொது விசாரணைகளில் எவ்வாறு பங்கேற்பது** — சுற்றுச்சூழல் தாக்க மதிப்பீடுகளுக்கான

---

## வார்டு அளவிலான பொறுப்புணர்வு சுருக்கம்

தளத்தின் AI உதவி சுருக்க உருவாக்கி (Google Gemini மூலம் இயக்கப்படுகிறது) எந்த நகரத்திற்கும் தனிப்பயனாக்கப்பட்ட பொறுப்புணர்வு சுருக்கத்தை உருவாக்குகிறது, வார்டு கவுன்சிலர்கள், குடியிருப்பு நல சங்கங்கள் அல்லது பத்திரிகையாளர்களுக்கு ஏற்றதாக. இதில் உள்ளடங்கியவை:

- நகரத்திற்கான தற்போதைய AQI அளவீடுகள்
- NCAP இலக்குகளுடன் ஒப்பீடு
- முக்கிய உள்ளூர் மாசு மூலங்கள்
- தேர்ந்தெடுக்கப்பட்ட அதிகாரிகளிடம் கேட்க பரிந்துரைக்கப்பட்ட கேள்விகள்

சுருக்கத்தை உருவாக்க, API endpoint-ஐ பயன்படுத்தவும்: `GET /.netlify/functions/accountability-brief?city={cityKey}`

## முகமூடி தேர்வு வழிகாட்டி

வெவ்வேறு AQI அளவுகளுக்கு சரியான முகமூடியை தேர்ந்தெடுப்பதற்கான நடைமுறை வழிகாட்டி:

| AQI அளவு | பரிந்துரைக்கப்பட்ட பாதுகாப்பு |
|-----------|----------------------|
| நல்லது–மிதமானது | முகமூடி தேவையில்லை |
| உணர்திறன் குழுக்களுக்கு ஆரோக்கியமற்றது | உணர்திறன் நபர்களுக்கு N95/KN95 |
| ஆரோக்கியமற்றது | அனைவருக்கும் N95/KN95 பரிந்துரை |
| மிகவும் ஆரோக்கியமற்றது | N95/KN95, வெளிப்புற நேரத்தைக் கட்டுப்படுத்துங்கள் |
| அபாயகரமானது | வீட்டிற்குள் இருங்கள்; வெளியே சென்றால் N95 |

---

## உட்புற காற்று தரம்

உட்புற காற்று தரப் பிரிவு உள்ளடக்குகிறது:

- வெவ்வேறு கட்டிட வகைகளுக்கான காற்றோட்ட உத்திகள்
- காற்று சுத்திகரிப்பான் தேர்வு (HEPA வடிகட்டி தரநிலைகள், அறை அளவுக்கான CADR மதிப்பீடுகள்)
- சமையல் எரிபொருள்கள் மற்றும் சுகாதாரம் — LPG vs திடஎரிபொருள்கள், உஜ்வாலா திட்ட அடைவு
- குறைந்த செலவு உணரிகளுடன் உட்புற காற்று தரத்தை கண்காணித்தல்
