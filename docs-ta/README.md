# JanVayu ஆவணங்கள்

**JanVayu** (जनवायु — "மக்களின் காற்று") என்பது இந்தியாவின் சுதந்திரமான, குடிமக்கள் தலைமையிலான காற்று தர பொறுப்புணர்வு தளமாகும். இது 30+ நகரங்களின் நேரடி மாசு தரவுகளை கண்காணிக்கிறது, சுகாதார தாக்கங்களை அளவிடுகிறது, NCAP பட்ஜெட் செலவுகளை பின்தொடர்கிறது, மேலும் அரசாங்கங்களை பொறுப்பேற்கச் செய்கிறது.

> **நேரடி தளம்:** [www.janvayu.in](https://www.janvayu.in)
> **GitHub:** [github.com/JanVayu/JanVayu](https://github.com/JanVayu/JanVayu)

---

## JanVayu என்றால் என்ன?

இந்தியாவில் ஆண்டுதோறும் 1.7 முதல் 2 மில்லியன் மக்கள் காற்று மாசுபாட்டால் இறக்கின்றனர். PM2.5 வெளிப்பாடு சராசரி இந்திய ஆயுளை 3.5 ஆண்டுகள் குறைக்கிறது. டெல்லியின் ஆண்டு PM2.5 சுமார் ~100 µg/m³ ஆகும் — இது WHO வழிகாட்டுதலை விட 20 மடங்கு அதிகம். பொருளாதார செலவு ஆண்டுக்கு $150 பில்லியனுக்கு மேல்.

JanVayu ஒரு பிரச்சாரம் அல்ல. இது ஒரு பதிவு — காற்று தர நெருக்கடி, அதன் தரவுகள், அதன் பாதிக்கப்பட்டவர்கள், அதன் கொள்கைகள் மற்றும் அதன் பொது நினைவகத்தை ஆவணப்படுத்தும் நிரந்தர பொது காப்பகம்.
