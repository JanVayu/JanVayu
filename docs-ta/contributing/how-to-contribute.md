# எவ்வாறு பங்களிப்பது

இந்தியாவின் காற்று தர நெருக்கடியை விரிவாக ஆவணப்படுத்த JanVayu சமூக பங்கேற்பை சார்ந்துள்ளது. ஆராய்ச்சியாளர்கள், பத்திரிகையாளர்கள், டெவலப்பர்கள் மற்றும் குடிமக்கள் அனைவரின் பங்களிப்புகளும் வரவேற்கப்படுகின்றன.

---

## பங்களிப்பு வகைகள்

### ஆராய்ச்சியாளர்கள்
- காற்று மாசுபாட்டு சுகாதார தாக்கங்கள் பற்றிய datasets மற்றும் கட்டுரைகளை பகிரவும்
- ஏற்கனவே உள்ள உள்ளடக்கத்தை சரிபார்க்கவும் உண்மை சோதிக்கவும் உதவுங்கள்
- பகுப்பாய்வு மற்றும் காட்சிப்படுத்தல்களை பங்களிக்கவும்
- ஆராய்ச்சி சுருக்கங்களை இந்திய மொழிகளுக்கு மொழிபெயர்க்கவும்

### பத்திரிகையாளர்கள்
- சரிபார்க்கப்பட்ட ஆய்வு அறிக்கைகளை பகிரவும்
- உள்ளூர் காற்று தர கதைகளை ஆவணப்படுத்தவும்
- முக்கிய ஊடக கவரேஜை காப்பகப்படுத்தவும்
- பொறுப்புணர்வு கண்காணிப்பானுக்கு பங்களிக்கவும்

### டெவலப்பர்கள்
- பிழைகளை அறிக்கை செய்யவும் மேம்பாடுகளை பரிந்துரைக்கவும்
- குறியீடு திருத்தங்கள் மற்றும் புதிய அம்சங்களை பங்களிக்கவும்
- தரவு காட்சிப்படுத்தல் கருவிகளை உருவாக்கவும்
- அணுகல்தன்மை மற்றும் செயல்திறனை மேம்படுத்தவும்

### குடிமக்கள்
- நேரடி சாட்சியங்களை பகிரவும் (விரும்பினால் பெயரற்றதாக)
- உள்ளூர் காற்று தர நிலைமைகளை ஆவணப்படுத்தவும்
- உள்ளடக்கத்தை பிராந்திய மொழிகளுக்கு மொழிபெயர்க்க உதவுங்கள்
- விழிப்புணர்வை பரப்புங்கள்

### வடிவமைப்பாளர்கள்
- காட்சி அணுகல்தன்மையை மேம்படுத்தவும்
- Infographics மற்றும் கல்வி பொருட்களை உருவாக்கவும்
- UI/UX மேம்பாடுகளுக்கு உதவுங்கள்

---

## குறியீடு பங்களிப்புகளுக்கு

```bash
# 1. Repository-ஐ fork செய்யுங்கள்
# 2. உங்கள் fork-ஐ clone செய்யுங்கள்
git clone https://github.com/YOUR_USERNAME/JanVayu.git
cd JanVayu

# 3. ஒரு கிளையை உருவாக்குங்கள்
git checkout -b feature/your-feature-name

# 4. மாற்றங்களை செய்து உள்ளூரில் சோதிக்கவும்
netlify dev

# 5. தெளிவான message-உடன் commit செய்யுங்கள்
git commit -m "feat(dashboard): add PM10 toggle to city cards"

# 6. Push செய்து Pull Request திறக்கவும்
git push origin feature/your-feature-name
```

[github.com/Varnasr/JanVayu](https://github.com/Varnasr/JanVayu)-ல் `main` கிளைக்கு எதிராக PR திறக்கவும்.

## உள்ளடக்க பங்களிப்புகளுக்கு

1. நீங்கள் என்ன பங்களிக்க விரும்புகிறீர்கள் என்று விவரிக்கும் GitHub-ல் **Issue திறக்கவும்**
2. **மூலங்களை வழங்குங்கள்** — அனைத்து உள்ளடக்கமும் URL அல்லது ஆவணத்துடன் சரிபார்க்கக்கூடியதாக இருக்க வேண்டும்
3. **சூழலை சேர்க்கவும்** — JanVayu காப்பகத்திற்கு இது ஏன் தொடர்புடையது என்று விளக்குங்கள்
4. அங்கீகரிக்கப்பட்டால், பராமரிப்பாளர் அதை சேர்ப்பதில் வழிகாட்டுவார்

## தரவு பங்களிப்புகளுக்கு

- தரவு நம்பகமான, சரிபார்க்கக்கூடிய மூலத்திலிருந்து வருவதை உறுதிப்படுத்தவும்
- தெளிவான பண்புக்கூறு மற்றும் உரிம தகவலை வழங்கவும்
- பொருந்தினால் முறையியல் ஆவணங்களை சேர்க்கவும்
- RTI பதில்கள் அல்லது அசல் ஆவணங்களை இணைக்கவும்

## Pull Request சரிபார்ப்பு பட்டியல்

PR சமர்ப்பிப்பதற்கு முன், சரிபார்க்கவும்:

- [ ] மாற்றங்கள் உள்ளூர் மேம்பாட்டில் (`netlify dev`) சரியாக வேலை செய்கின்றன
- [ ] எந்த கோப்பிலும் ரகசியங்கள் அல்லது API keys hardcode செய்யப்படவில்லை
- [ ] புதிய சூழல் மாறிகள் `docs/technical/environment-variables.md`-ல் ஆவணப்படுத்தப்பட்டுள்ளன
- [ ] குறியீடு [குறியீடு தரநிலைகளை](code-standards.md) பின்பற்றுகிறது
- [ ] உள்ளடக்கம் [உள்ளடக்க தரநிலைகளை](content-standards.md) பின்பற்றுகிறது
- [ ] Mobile-ல் site சரியாக render ஆகிறது (browser devtools-ல் சோதிக்கவும்)
