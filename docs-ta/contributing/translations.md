# மொழிபெயர்ப்புகள்

இந்தியாவின் மொழி பன்முகத்தன்மையில் அணுகல்தன்மைக்கு JanVayu உறுதிபூண்டுள்ளது. தளம் தற்போது ஆங்கிலம், ஹிந்தி, தமிழ், மராத்தி மற்றும் பெங்காலி ஆகிய மொழிகளை ஆதரிக்கிறது.

## மொழிபெயர்ப்பை எவ்வாறு பங்களிப்பது

### புதிய மொழி சேர்த்தல்

1. இந்த தலைப்பில் Issue திறக்கவும்: `Translation: Add [Language Name] support`
2. சேர்க்கவும்:
   - உங்கள் பெயர் (அல்லது handle) மற்றும் மொழி திறன்
   - காலப்போக்கில் மொழிபெயர்ப்புகளை பராமரிக்க முடியுமா (உள்ளடக்க புதுப்பிப்புகளுக்கு ஏற்ப)
3. பராமரிப்பாளர் ஒரு கிளையை உருவாக்கி, மொழிபெயர்க்க source strings-ஐ வழங்குவார்

### ஏற்கனவே உள்ள மொழிபெயர்ப்பை மேம்படுத்துதல்

தவறான அல்லது அசௌகரியமான மொழிபெயர்ப்பை கவனித்தால்:

1. இதனுடன் Issue திறக்கவும்: `Translation: Fix [Language Name] — [specific section/string]`
2. வழங்கவும்: தற்போதைய உரை, திருத்தப்பட்ட உரை மற்றும் சுருக்க விளக்கம்

## மொழிபெயர்ப்பு கொள்கைகள்

- **துல்லியம் மிகவும் முக்கியம்** — சொல்லுக்குச் சொல் அல்ல, இயற்கையான மொழியில் சரியான பொருளை கடத்தவும்
- **தொழில்நுட்ப சொற்களை பாதுகாக்கவும்** — PM2.5, AQI, NCAP, GEMM போன்ற சொற்கள் மாறாமல் இருக்க வேண்டும் (அவை சர்வதேச அளவில் தரப்படுத்தப்பட்டவை)
- **எளிய மொழி பயன்படுத்தவும்** — மொழிபெயர்ப்புகள் ஆங்கில பதிப்பின் அதே பார்வையாளர்களுக்கு அணுகக்கூடியதாக இருக்க வேண்டும்: படித்த பொது மக்கள், நிபுணர்கள் அல்ல
- **நிலைத்தன்மை** — ஒரே கருத்துக்கு ஒரே மொழிபெயர்க்கப்பட்ட சொல்லை முழுவதும் பயன்படுத்தவும் (எ.கா., "air quality index"-ஐ எப்போதும் ஒரே விதமாக மொழிபெயர்க்கவும்)
- **நடுநிலை தொனி** — JanVayu கட்சி சார்பற்றது; மொழிபெயர்ப்புகள் இந்த நடுநிலையை பாதுகாக்க வேண்டும்

## உணர்திறன் சொற்கள்

| ஆங்கிலம் | குறிப்புகள் |
|---------|-------|
| Non-Attainment City | தேசிய காற்று தர தரநிலைகளை "அடையவில்லை" (சந்திக்க தவறிய) நகரம் என்று மொழிபெயர்க்கவும் |
| Graded Response Action Plan (GRAP) | GRAP என்ற சுருக்கத்தைப் பயன்படுத்தவும்; முதல் பயன்பாட்டில் விரிவாக எழுதவும் |
| PM2.5 | அனைத்து மொழிகளிலும் PM2.5 ஆகவே வைக்கவும் |
| NCAP | NCAP ஆகவே வைக்கவும்; முதல் பயன்பாட்டில் National Clean Air Programme என்று விரிவாக எழுதவும் |
| CPCB | CPCB ஆகவே வைக்கவும்; முதல் பயன்பாட்டில் Central Pollution Control Board என்று விரிவாக எழுதவும் |
