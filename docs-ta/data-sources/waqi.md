# நிகழ்நேர AQI (WAQI)

JanVayu நிகழ்நேர AQI தரவின் முதன்மை மூலமாக [World Air Quality Index (WAQI)](https://waqi.info) API-ஐ பயன்படுத்துகிறது.

---

## WAQI பற்றி

WAQI என்பது 100+ நாடுகளில் 20,000+ கண்காணிப்பு நிலையங்களின் தரவை தொகுக்கும் சுதந்திர இலாப நோக்கமற்ற திட்டம், இந்தியாவில் உள்ள அனைத்து CPCB CAAQMS நிலையங்கள் உட்பட. இந்தியாவுக்கு, WAQI இவற்றின் தரவை தொகுக்கிறது:

- CPCB (Central Pollution Control Board) — அதிகாரபூர்வ அரசு கண்காணிப்பு network
- மாநில மாசுபாட்டு கட்டுப்பாட்டு வாரியங்கள்
- தூதரக கண்காணிப்பு (US Embassy, போன்றவை)

---

## JanVayu WAQI-ஐ எவ்வாறு பயன்படுத்துகிறது

WAQI API ஒவ்வொரு 10 நிமிடமும் **உலாவியிலிருந்து நேரடியாக** (கிளையண்ட்-பக்கம்) அழைக்கப்படுகிறது. API token `index.html`-ல் உட்பொதிக்கப்பட்ட free-tier public key ஆகும்.

```javascript
// கிளையண்ட்-பக்க AQI fetch (எளிமைப்படுத்தப்பட்டது)
const url = `https://api.waqi.info/feed/${cityStation}/?token=${WAQI_TOKEN}`;
const response = await fetch(url);
const data = await response.json();
```

இதன் பொருள்:
- நேரடி AQI-க்கு சர்வர்-பக்க உள்கட்டமைப்பு தேவையில்லை
- பக்கம் திறந்திருக்கும் போது தரவு தானாக புதுப்பிக்கிறது
- WAQI-ன் free-tier rate limits-க்கு அப்பால் எந்த செலவும் இல்லை

---

## AQI vs. PM2.5

WAQI AQI-ஐ US EPA அளவுகோலில் தெரிவிக்கிறது. JanVayu இரண்டையும் காட்டுகிறது:

- **AQI** — தரப்படுத்தப்பட்ட 0–500 index (US EPA அளவுகோல்)
- **PM2.5 (µg/m³)** — raw நுண்ணிய துகள் செறிவு

PM2.5 செறிவு மற்றும் AQI இடையேயான மாற்றம் US EPA breakpoints-ஐ பயன்படுத்துகிறது. இந்தியா சற்று வேறுபட்ட breakpoints-உடன் சொந்த National AQI (NAQI) அளவுகோலை பயன்படுத்துகிறது — தொடர்புள்ள இடங்களில் JanVayu இந்த வேறுபாட்டை குறிப்பிடுகிறது.

---

## Rate Limits

பொது WAQI token IP அளவில் rate-limited. உள்ளூரில் பல API அழைப்புகளை இயக்கினால் (எ.கா., 30 நகரங்களையும் ஒரே நேரத்தில் சோதித்தல்), rate limits-ஐ அடையலாம். விருப்பங்கள்:

1. **உங்கள் சொந்த WAQI token-ஐ பயன்படுத்தவும்** — [aqicn.org/data-platform/token](https://aqicn.org/data-platform/token/)-ல் இலவசமாக பதிவு செய்து `index.html`-ல் token-ஐ மாற்றவும்
2. **கோரிக்கைகளை மெதுவாக்கவும்** — மேம்பாட்டில் நகர fetch-களுக்கு இடையே சிறிய தாமதத்தை சேர்க்கவும்

---

## நிலைய கவரேஜ்

இந்தியாவுக்கான WAQI கவரேஜ் முக்கிய பெருநகரங்கள் மற்றும் மாநில தலைநகரங்களில் வலுவாக உள்ளது. கவரேஜ் குறைவாக உள்ளவை:

- சிறிய மாவட்ட நகரங்கள்
- கிராமப்புறங்கள்
- வடகிழக்கு மாநிலங்கள் (குறைவான CPCB நிலைய இருப்பு)

தரை கண்காணிப்பு இல்லாத இடங்களில், JanVayu வரம்பை குறிப்பிடுகிறது மற்றும் செயற்கைக்கோள் வழி மதிப்பீடுகளை குறிப்பிடலாம்.
