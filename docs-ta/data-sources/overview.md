# தரவு மூலங்கள் கண்ணோட்டம்

JanVayu 160+ சரிபார்க்கப்பட்ட பொது தரவு மூலங்களை ஒருங்கிணைக்கிறது. தளத்தில் உள்ள ஒவ்வொரு தரவு புள்ளியும் முதன்மை மூலத்திற்கு கண்டுபிடிக்கக்கூடியது. இந்தப் பக்கம் முக்கிய வகைகளை பட்டியலிடுகிறது.

## மேற்கோள் மற்றும் பண்புக்கூறு

JanVayu எப்போதும் முதன்மை மூலங்களுக்கு இணைக்கிறது. JanVayu-லிருந்து தரவை மீண்டும் உருவாக்கும்போது:

- அசல் மூலத்தை (Lancet, CPCB, போன்றவை) மேற்கோள் காட்டுங்கள், JanVayu-ஐ அல்ல
- JanVayu ஒரு தொகுப்பான் மற்றும் பொறுப்புணர்வு தளம், முதன்மை தரவு உற்பத்தியாளர் அல்ல
- முறையியல் கேள்விகளுக்கு, அசல் ஆராய்ச்சி கட்டுரைகளைக் குறிப்பிடவும்
