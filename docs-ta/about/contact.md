## பாதுகாப்பு சிக்கலை அறிக்கை செய்தல்

பாதுகாப்பு பாதிப்பை கண்டறிந்தால், **பொது GitHub Issue திறக்க வேண்டாம்**. அதற்கு பதிலாக, [SECURITY.md](https://github.com/Varnasr/JanVayu/blob/main/SECURITY.md)-ல் விவரிக்கப்பட்ட responsible disclosure process-ஐ பின்பற்றவும்.

---

## ஊடகம் மற்றும் ஆராய்ச்சி விசாரணைகள்

பத்திரிகையாளர்கள், ஆராய்ச்சியாளர்கள் மற்றும் கொள்கை நிபுணர்கள் JanVayu-ன் தரவு மற்றும் பகுப்பாய்வை பயன்படுத்த வரவேற்கிறோம். தயவுசெய்து:

1. குறிப்பிட்ட புள்ளிவிவரங்களுக்கு JanVayu-ஐ அல்ல, **முதன்மை மூலத்தை** (CPCB, Lancet, போன்றவை) மேற்கோள் காட்டவும்
2. தொகுக்கப்பட்ட தரவை அணுகிய தளமாக JanVayu-ஐ குறிப்பிடவும்
3. தனிப்பயன் பகுப்பாய்வு அல்லது தரவு கோரிக்கைகளுக்கு, GitHub Issue திறக்கவும் அல்லது [contribute@janvayu.in](mailto:contribute@janvayu.in)-க்கு மின்னஞ்சல் செய்யவும்
